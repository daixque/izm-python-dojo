Your First Program

In the programming world, it's a tradition to display 'Hello, World!' as your first program when learning a new language.

Through this simple program, you can experience the basic flow of writing and running code.

What is the print Function?

In Python, you can use the print() function to display text and numbers on the screen.

When you write what you want to display inside the parentheses, that content will appear in the output.

How to Write Strings

When displaying strings (text), you need to enclose them in double quotes (") or single quotes (').

Either works the same way, but it's recommended to use them consistently.

Examples

Basic Usage
print("Hello, World!")
Display a string enclosed in double quotes.
Single Quotes Work Too
print('Hello, Python!')
Single quotes work the same way.
Display Multiple Lines
print("Hello")
print("Python!")
Multiple print statements will each display on a new line.
Display Numbers
print(123)
print(3 + 5)
Numbers can be displayed without quotes. You can also display calculation results.
