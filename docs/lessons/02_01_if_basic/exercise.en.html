📋 Task

Create a program that asks the user to enter a number, then determines if it's positive, negative, or zero.

Examples:

  • Input: 5 → Output: Positive number
  • Input: -3 → Output: Negative number
  • Input: 0 → Output: Zero

✏️ Instructions

  • Use input() to get a number and convert it to integer with int()
  • Use if number > 0: to check if it's positive
  • Use if number < 0: to check if it's negative
  • Use if number == 0: to check if it's zero

▶ 💡 Hints

  • First, use number = int(input("Enter a number: ")) to get input
  • Then write if number > 0: with the positive number logic
  • Similarly, add if number < 0: and if number == 0:
  • Use print() after each condition to display the result
