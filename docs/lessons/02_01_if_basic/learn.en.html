🎯 What are Conditional Statements?

Programs need to perform different actions based on different situations. For example:

  • If the test score is 60 or above, display "Pass"
  • If it's raining, display "Bring an umbrella"
  • If age is 18 or above, display "You're an adult"

This kind of "if... then..." logic is called conditional branching.

📝 Basic if Statement Syntax

In Python, we use the if statement for conditional branching.

Basic if statement
age = 15

if age >= 18:
    print("You are an adult")

print("Program finished")
Syntax explanation:
  • if condition: - Executes only when the condition is True
  • A colon : is required at the end of the line
  • Code to be executed must be indented
  • If the condition is False, the indented lines are skipped

🔍 Comparison Operators

We use comparison operators to create conditions.

Operator Meaning Example Result
== Equal to 5 == 5 True
!= Not equal to 5 != 3 True
> Greater than 10 > 5 True
< Less than 3 < 5 True
>= Greater than or equal 5 >= 5 True
<= Less than or equal 3 <= 5 True
Comparison operator examples
score = 85

if score >= 60:
    print("You passed!")

if score == 100:
    print("Perfect score!")

temperature = 30

if temperature > 25:
    print("It's hot today")
You can use multiple conditions to make different judgments.

⚠️ The Importance of Indentation

Indentation is extremely important in Python.

Correct indentation
age = 15

if age >= 18:
    print("You are an adult")
    print("You can vote")

print("This always runs")
Code inside an if statement must be indented with 4 spaces.
Wrong indentation (Error)
age = 15

if age >= 18:
print("You are an adult") # ← Error! No indentation
Without indentation, you'll get an error.

Key points:

  • Use 4 spaces or 1 tab for indentation
  • Code inside an if statement must be indented
  • Code in the same block must have the same indentation level

✨ True and False

Comparison operators return boolean values.

Checking boolean values
# Store comparison results in variables
result = 10 > 5
print(result) # True

result2 = 3 > 10
print(result2) # False

# if statements check boolean values
is_adult = True

if is_adult:
    print("You are an adult")
Comparison results are either True or False.

🎯 Summary

  • Use if condition: for conditional branching
  • Create conditions with comparison operators (==, !=, >, <, >=, <=)
  • Indented code runs only when the condition is True
  • Always use 4 spaces or 1 tab for indentation consistently
