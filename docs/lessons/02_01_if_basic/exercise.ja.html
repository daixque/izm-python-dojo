📋 課題内容

ユーザーから数値を入力してもらい、その数値が正の数か、負の数か、0かを判定するプログラムを作りましょう。

例：

  • 入力: 5 → 出力: 正の数です
  • 入力: -3 → 出力: 負の数です
  • 入力: 0 → 出力: 0です

✏️ やること

  • input() で数値を入力してもらい、int() で整数に変換します
  • if number > 0: で正の数かチェックします
  • if number < 0: で負の数かチェックします
  • if number == 0: で0かチェックします

▶ 💡 ヒント

  • まず number = int(input("数値を入力してください: ")) で入力を受け取ります
  • 次に if number > 0: で正の数の時の処理を書きます
  • 同様に if number < 0: と if number == 0: も書きます
  • それぞれの条件の後に print() で結果を表示します
