📋 Task

Implement the function linear_search(items, target).

Search the list items for target using linear search (checking each element from the beginning). Return the index of the target if found, or -1 if not found.

Example Output

  print(linear_search([3, 1, 4, 1, 5], 4))  # 2
print(linear_search([3, 1, 4, 1, 5], 9))  # -1
print(linear_search([3, 1, 4, 1, 5], 1))  # 1 (first occurrence)

✏️ Instructions

  • Use a for loop with range(len(items)) to check each index i
  • If items[i] == target, return i
  • If the loop finishes without finding the target, return -1

▶ 💡 Hints

  • Loop through indices and compare each element:
    def linear_search(items, target):
        for i in range(len(items)):
            if items[i] == ???:
                return i
  • After the loop, return -1 if not found:
    def linear_search(items, target):
        for i in range(len(items)):
            if items[i] == target:
                return i
        return ???
  • The completed function looks like this:
    def linear_search(items, target):
        for i in range(len(items)):
            if items[i] == target:
                return i
        return -1
