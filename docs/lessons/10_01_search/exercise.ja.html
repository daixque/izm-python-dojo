📋 課題内容

関数 linear_search(items, target) を実装しましょう。

リスト items の中から target を線形探索（先頭から順番に調べる）で探し、見つかった場合はそのインデックスを返します。見つからない場合は -1 を返します。

実行例

  print(linear_search([3, 1, 4, 1, 5], 4))  # 2
print(linear_search([3, 1, 4, 1, 5], 9))  # -1
print(linear_search([3, 1, 4, 1, 5], 1))  # 1 (最初に見つかった位置)

✏️ やること

  • for ループで range(len(items)) を使い、インデックス i を順番に調べる
  • items[i] == target なら i を返す
  • ループが終わっても見つからなかった場合は -1 を返す

▶ 💡 ヒント

  • ループでインデックスを使いながら調べます：
    def linear_search(items, target):
        for i in range(len(items)):
            if items[i] == ???:
                return i
  • ループの後、見つからなかった場合に -1 を返します：
    def linear_search(items, target):
        for i in range(len(items)):
            if items[i] == target:
                return i
        return ???
  • 完成した関数はこうなります：
    def linear_search(items, target):
        for i in range(len(items)):
            if items[i] == target:
                return i
        return -1
