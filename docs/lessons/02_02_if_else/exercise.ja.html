📋 課題内容

じゃんけんゲームのプログラムを完成させましょう。

コンピュータとじゃんけんをして、勝敗を判定するプログラムを作ります。

勝敗のルール

  • グー（1）はチョキ（2）に勝つ
  • チョキ（2）はパー（3）に勝つ
  • パー（3）はグー（1）に勝つ
  • 同じ手の場合は引き分け

✏️ やること

  • ユーザーに手を入力してもらう（1: グー、2: チョキ、3: パー）
  • コンピュータの手は既にランダムで生成されています
  • judge_winner(user, computer) 関数の中に勝敗判定のロジックを書く
  • 関数は "あなたの勝ち！"、"コンピュータの勝ち！"、"引き分け！" のいずれかを返します
  • elif を使って、すべての勝ちパターンをチェックしましょう
  • テスト後、実際に遊びたい場合は下部のコメントを外してください
