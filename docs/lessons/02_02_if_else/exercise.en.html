📋 Task

Complete a rock-paper-scissors game program.

Create a program that plays rock-paper-scissors with the computer and determines the winner.

Game Rules

  • Rock (1) beats Scissors (2)
  • Scissors (2) beats Paper (3)
  • Paper (3) beats Rock (1)
  • Same choice results in a tie

✏️ Instructions

  • Get the user's choice (1: Rock, 2: Scissors, 3: Paper)
  • The computer's choice is already randomly generated
  • Write the logic inside the judge_winner(user, computer) function
  • The function must return "You win!", "Computer wins!", or "It's a tie!"
  • Use elif to check all winning patterns
  • After testing, uncomment the lines at the bottom if you want to play the game
