🔀 The else Statement

In the previous lesson, we learned about the if statement. Now we'll learn how to handle cases when the condition is not met.

The else statement lets you specify what to do when the if condition is False.

Basic else Statement
age = 15

if age >= 18:
    print("Adult")
else:
    print("Child")
In this example, if age is not 18 or greater (i.e., less than 18), "Child" will be printed.

🔀 The elif Statement - Multiple Conditions

elif (short for "else if") allows you to check multiple conditions in sequence.

Using elif for Multiple Conditions
score = 75

if score >= 90:
    print("Excellent!")
elif score >= 70:
    print("Good")
elif score >= 60:
    print("Pass")
else:
    print("Fail")
This code checks conditions from top to bottom:
  • First checks score >= 90 (False)
  • Next checks score >= 70 (True!) → Prints "Good" and stops
  • Remaining conditions are not checked

⚠️ Condition Priority

Important: elif statements are evaluated from top to bottom, and only the first block with a True condition is executed.

❌ Incorrect Example
if score >= 60:
    print("Pass")
elif score >= 90:
    print("Excellent!") # This will never be reached!
A student with 90 points will only see "Pass", not "Excellent!". This is because the first condition score >= 60 is True, so execution stops there.
✓ Correct Example
if score >= 90:
    print("Excellent!")
elif score >= 60:
    print("Pass")
Tip: Write more restrictive conditions (higher values) first!

📊 Flowcharts

A diagram showing the flow of conditional logic is called a "flowchart".

flowchart TD Start([Start]) --> Cond1{Condition 1} Cond1 -->|Yes| ProcessA[Process A] Cond1 -->|No| Cond2{Condition 2} Cond2 -->|Yes| ProcessB[Process B] Cond2 -->|No| ElseBlock[else block] ProcessA --> End([End]) ProcessB --> End ElseBlock --> End

Drawing such diagrams before writing code helps organize complex conditional logic.

🎲 Random Numbers - The random Module

Games and simulations often need random numbers. Python provides the random module for this.

Generating Random Integers
import random

# Generate a random integer from 1 to 6
dice = random.randint(1, 6)
print(f"Dice roll: {dice}")
random.randint(a, b) returns a random integer between a and b (inclusive).
Note: import random should be written once at the beginning of your program.
Rock-Paper-Scissors Example
import random

# 1:Rock, 2:Scissors, 3:Paper
computer = random.randint(1, 3)

if computer == 1:
    print("Computer: Rock")
elif computer == 2:
    print("Computer: Scissors")
else:
    print("Computer: Paper")
Different outputs are produced based on the randomly generated number.

🎯 Summary

  • else: Executes when the if condition is False
  • elif: Allows checking multiple conditions in sequence
  • Conditions are evaluated top to bottom; only the first True block executes
  • Write more restrictive conditions first
  • random.randint(a, b) generates a random integer
