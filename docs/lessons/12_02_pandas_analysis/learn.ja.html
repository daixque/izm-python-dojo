データ集計とフィルタリング

📚 説明
English

🔎 条件でデータを絞り込む（フィルタリング）

Pandasでは、条件に合う行だけを取り出すことができます。これをフィルタリングと言います。

例えば「点数が70点以上の生徒だけを取り出す」といった操作が簡単にできます。

条件フィルタリングの基本
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "Dave"],
    "score": [80, 55, 90, 45]
})

# 点数が70以上の行だけを取り出す
passing = df[df["score"] >= 70]
print(passing)
# name score
# 0 Alice 80
# 2 Charlie 90
df[df["score"] >= 70] で、score列が70以上の行だけを抽出できます。
df["score"] >= 70 の部分がTrue/Falseの条件式になり、Trueの行だけが残ります。
複数条件のフィルタリング
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "Dave"],
    "score": [80, 55, 90, 45],
    "class": ["A", "B", "A", "B"]
})

# Aクラスかつ点数が70以上
result = df[(df["class"] == "A") & (df["score"] >= 70)]
print(result)
複数条件を組み合わせるときは &（AND）や |（OR）を使います。
各条件を () で囲むのがポイントです。

📦 groupby()でグループ集計

groupby() を使うと、特定の列の値でデータをグループ分けして集計できます。

例えば「クラスごとの平均点」や「チームごとの合計得点」などが計算できます。

グループごとの集計
import pandas as pd

df = pd.DataFrame({
    "class": ["A", "B", "A", "B", "A"],
    "score": [80, 60, 90, 70, 85]
})

# クラスごとの平均点
group_avg = df.groupby("class")["score"].mean()
print(group_avg)
# class
# A 85.0
# B 65.0
groupby("class") でクラスごとにグループを作り、
["score"].mean() でそれぞれのグループの平均点を計算します。

🔃 sort_values()で並び替え

sort_values() を使うと、指定した列の値でデータを並び替えられます。

昇順・降順に並び替え
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, 55, 90]
})

# 点数の低い順（昇順）
asc = df.sort_values("score")
print(asc)

# 点数の高い順（降順）
desc = df.sort_values("score", ascending=False)
print(desc)
デフォルトは昇順（小さい順）です。
ascending=False を指定すると降順（大きい順）になります。

❓ 欠損値（NaN）の扱い

実際のデータには、値が抜けていること（欠損値）がよくあります。

Pandasでは欠損値を NaN（Not a Number）として扱います。

NaNを含むデータの処理
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, np.nan, 90]
})

# NaNを含む行を確認
print(df.isnull())

# NaNを除いた平均（自動でNaNを無視する）
print(df["score"].mean())  # 85.0
isnull() でどこにNaNがあるか確認できます。
mean() などの集計関数は自動でNaNを無視して計算してくれます。
NaNを除く・埋める
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "score": [80, np.nan, 90, np.nan, 70]
})

# NaNの行を削除
clean = df.dropna()
print(clean)

# NaNを0で埋める
filled = df.fillna(0)
print(filled)
dropna()：NaNを含む行を削除します。
fillna(値)：NaNを指定した値で置き換えます。

💡 まとめ

  • df[df["列名"] 条件] で条件に合う行だけを取り出せる
  • 複数条件は &（AND）や |（OR）で組み合わせる
  • groupby("列名") でグループごとの集計ができる
  • sort_values("列名") でデータを並び替えられる
  • 欠損値（NaN）は mean() などで自動的に無視される
  • dropna() や fillna() で欠損値を処理できる
