🔎 Filtering Data with Conditions

In Pandas, you can extract only the rows that match a condition. This is called filtering.

For example, you can easily pull out "only students who scored 70 or above".

Basic Conditional Filtering
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "Dave"],
    "score": [80, 55, 90, 45]
})

# Keep only rows where score >= 70
passing = df[df["score"] >= 70]
print(passing)
# name score
# 0 Alice 80
# 2 Charlie 90
df[df["score"] >= 70] extracts rows where the score column is 70 or more.
df["score"] >= 70 creates a True/False condition, and only the True rows are kept.
Multiple Conditions
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "Dave"],
    "score": [80, 55, 90, 45],
    "class": ["A", "B", "A", "B"]
})

# Class A AND score >= 70
result = df[(df["class"] == "A") & (df["score"] >= 70)]
print(result)
Use & (AND) or | (OR) to combine multiple conditions.
Make sure to wrap each condition in parentheses ().

📦 Grouping Data with groupby()

groupby() lets you group data by the values in a column and then aggregate each group.

For example, you can calculate "average score per class" or "total points per team".

Aggregating by Group
import pandas as pd

df = pd.DataFrame({
    "class": ["A", "B", "A", "B", "A"],
    "score": [80, 60, 90, 70, 85]
})

# Average score per class
group_avg = df.groupby("class")["score"].mean()
print(group_avg)
# class
# A 85.0
# B 65.0
groupby("class") creates groups based on the class column.
["score"].mean() then calculates the average score for each group.

🔃 Sorting Data with sort_values()

sort_values() lets you sort data by the values in a specified column.

Ascending and Descending Order
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, 55, 90]
})

# Sort ascending (lowest first)
asc = df.sort_values("score")
print(asc)

# Sort descending (highest first)
desc = df.sort_values("score", ascending=False)
print(desc)
The default sort order is ascending (smallest to largest).
Set ascending=False to sort in descending order (largest to smallest).

❓ Handling Missing Values (NaN)

Real-world data often has missing entries. Pandas represents missing values as NaN (Not a Number).

Working with NaN Data
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, np.nan, 90]
})

# Check where NaN values are
print(df.isnull())

# Mean ignoring NaN automatically
print(df["score"].mean())  # 85.0
isnull() shows where NaN values are located.
Aggregation functions like mean() automatically ignore NaN values.
Removing or Filling NaN
import pandas as pd
import numpy as np

df = pd.DataFrame({
    "score": [80, np.nan, 90, np.nan, 70]
})

# Remove rows with NaN
clean = df.dropna()
print(clean)

# Fill NaN with 0
filled = df.fillna(0)
print(filled)
dropna(): Removes rows that contain NaN values.
fillna(value): Replaces NaN with the specified value.

💡 Summary

  • Use df[df["column"] condition] to filter rows matching a condition
  • Combine multiple conditions with & (AND) or | (OR)
  • Use groupby("column") to aggregate data by group
  • Use sort_values("column") to sort data
  • NaN (missing values) are automatically ignored by functions like mean()
  • Use dropna() or fillna() to handle missing values
