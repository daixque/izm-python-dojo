📋 課題内容

関数 passing_average(df, score_col, threshold=60) を実装しましょう。

指定した列で threshold（しきい値）以上のスコアを持つ行だけを取り出し、その平均値を返します。

引数

  • df: PandasのDataFrame
  • score_col: スコアが入っている列名（文字列）
  • threshold: 合格ラインのスコア（デフォルトは 60）

実行例

  import pandas as pd
df = pd.DataFrame({"score": [50, 70, 80, 40, 90]})
print(passing_average(df, "score", threshold=60))  # 80.0
# 70, 80, 90 の平均 = 80.0

✏️ やること

  • import pandas as pd を先頭に書いてください
  • df[df[score_col] >= threshold] で合格点以上の行を取り出します
  • 絞り込んだデータの score_col 列の平均（.mean()）を返します
  • デフォルト引数 threshold=60 を関数のシグネチャに含めてください

▶ 💡 ヒント

  • 条件フィルタリングの書き方：
    passing = df[df[score_col] >= threshold]
    これでthreshold以上のスコアを持つ行だけが残ります。
  • フィルタリングした後で平均を計算します：
    passing[score_col].mean()
  • 完成形はこのようになります：
    def passing_average(df, score_col, threshold=60):
        passing = df[df[score_col] >= threshold]
        return passing[score_col].mean()
