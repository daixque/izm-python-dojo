📋 Task

Implement the function passing_average(df, score_col, threshold=60).

It filters rows where the score in score_col is greater than or equal to threshold, then returns the mean of those scores.

Arguments

  • df: A Pandas DataFrame
  • score_col: The name of the column containing scores (string)
  • threshold: The minimum passing score (default is 60)

Example

  import pandas as pd
df = pd.DataFrame({"score": [50, 70, 80, 40, 90]})
print(passing_average(df, "score", threshold=60))  # 80.0
# Mean of 70, 80, 90 = 80.0

✏️ Instructions

  • Write import pandas as pd at the top of your code
  • Use df[df[score_col] >= threshold] to filter rows at or above the threshold
  • Return the mean of score_col in the filtered data using .mean()
  • Include the default argument threshold=60 in the function signature

▶ 💡 Hints

  • How to write conditional filtering:
    passing = df[df[score_col] >= threshold]
    This keeps only rows where the score meets or exceeds the threshold.
  • After filtering, calculate the mean:
    passing[score_col].mean()
  • The complete solution looks like this:
    def passing_average(df, score_col, threshold=60):
        passing = df[df[score_col] >= threshold]
        return passing[score_col].mean()
