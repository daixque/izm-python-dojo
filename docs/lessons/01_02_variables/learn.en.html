What are Variables?

Variables are like 'boxes' for storing data.

You can give these boxes names, and later use those names to retrieve the data or use it in calculations.

For example, if you put the number '14' in a box called 'age', you can later use that number by referring to 'age'.

Variable Assignment

Putting data into a variable is called 'assignment'. In Python, we use = (equals sign) for assignment.

= doesn't mean 'equal to'; it means 'put the value on the right into the variable on the left'.

You can choose variable names freely, but it's important to use clear and meaningful names.

Arithmetic Operations

In Python, you can do calculations just like in math.

  • + : Addition (e.g., 5 + 3 → 8)
  • - : Subtraction (e.g., 10 - 4 → 6)
  • * : Multiplication (e.g., 3 * 4 → 12)
  • / : Division (e.g., 15 / 3 → 5.0)

You can use numbers stored in variables for calculations too.

Basic Data Types

Python has different types of data.

  • Integer (int): Numbers without decimal points (e.g., 10, -5, 0)
  • Float (float): Numbers with decimal points (e.g., 3.14, -0.5, 2.0)
  • String (str): Text or characters (e.g., "Hello", "Python")

The result of division (/) is always a float.

Examples

Assign a Number to a Variable
age = 14
print(age)
Assign 14 to variable age and display it.
Calculate Using Variables
x = 10
y = 3
print(x + y)
Calculate and display the sum of two variables. The result is 13.
Arithmetic Operations Examples
a = 20
b = 4
print(a + b) # Addition → 24
print(a - b) # Subtraction → 16
print(a * b) # Multiplication → 80
print(a / b) # Division → 5.0
You can perform various calculations.
Store Calculation Results in a Variable
price = 150
count = 3
total = price * count
print(total)
You can store calculation results in a new variable. The result is 450.
Update a Variable's Value
score = 80
score = score + 10
print(score)
You can change a variable's value later. The result is 90.
Calculate with Decimals
height = 1.65
weight = 55.5
print(weight / height)
You can calculate with decimal numbers too.
