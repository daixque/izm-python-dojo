変数と計算

📚 説明
English

変数とは

変数は、データを保存しておくための「箱」のようなものです。

この箱には名前をつけることができ、後でその名前を使ってデータを取り出したり、計算に使ったりできます。

例えば、「年齢」という箱に「14」という数字を入れておくと、後で「年齢」という名前を使ってその数字を使うことができます。

変数への代入

変数にデータを入れることを「代入」と言います。Pythonでは=（イコール）を使って代入します。

=は「等しい」という意味ではなく、「右側の値を左側の変数に入れる」という意味です。

変数名は自分で自由に決められますが、わかりやすい名前をつけることが大切です。

四則演算

Pythonでは、算数と同じように計算ができます。

  • + : 足し算（例: 5 + 3 → 8）
  • - : 引き算（例: 10 - 4 → 6）
  • * : 掛け算（例: 3 * 4 → 12）
  • / : 割り算（例: 15 / 3 → 5.0）

変数に入れた数値も、同じように計算できます。

データ型の基本

Pythonには、いろいろな種類のデータがあります。

  • 整数（int）: 小数点のない数字（例: 10, -5, 0）
  • 小数（float）: 小数点のある数字（例: 3.14, -0.5, 2.0）
  • 文字列（str）: 文字や文章（例: "こんにちは", "Python"）

割り算（/）の結果は、必ず小数（float）になります。

コード例

変数に数値を代入
age = 14
print(age)
変数ageに14を代入して表示します。
変数を使った計算
x = 10
y = 3
print(x + y)
2つの変数の合計を計算して表示します。結果は13になります。
四則演算の例
a = 20
b = 4
print(a + b) # 足し算 → 24
print(a - b) # 引き算 → 16
print(a * b) # 掛け算 → 80
print(a / b) # 割り算 → 5.0
いろいろな計算をまとめて実行できます。
計算結果を変数に保存
price = 150
count = 3
total = price * count
print(total)
計算結果を新しい変数に保存できます。結果は450になります。
変数の値を更新
score = 80
score = score + 10
print(score)
変数の値は後から変更できます。結果は90になります。
小数の計算
height = 1.65
weight = 55.5
print(weight / height)
小数も計算できます。
