📋 Task

Implement the function factorial(n) using recursion.

It should return the factorial of n (n!), which is the product of all integers from 1 to n.

  • 0! = 1 (special rule)
  • 1! = 1
  • 5! = 5 × 4 × 3 × 2 × 1 = 120

Example Output

  print(factorial(0))   # 1
print(factorial(5))   # 120
print(factorial(10))  # 3628800

✏️ Instructions

  • Base case: when n == 0, return 1
  • Recursive case: return n * factorial(n - 1)
  • Use an if statement to separate the base case from the recursive case

▶ 💡 Hints

  • Start with the base case:
    def factorial(n):
        if n == 0:
            return ???
    What is the factorial of 0?
  • Now add the recursive case:
    def factorial(n):
        if n == 0:
            return 1
        return n * factorial(???)
    Since n! = n × (n-1)!, what should you pass to factorial?
  • The completed function looks like this:
    def factorial(n):
        if n == 0:
            return 1
        return n * factorial(n - 1)
