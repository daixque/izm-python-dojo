📋 課題内容

関数 factorial(n) を再帰を使って実装しましょう。

n の階乗（n!）を返します。階乗とは 1 から n までの整数をすべてかけ合わせた数です。

  • 0! = 1（特別なルール）
  • 1! = 1
  • 5! = 5 × 4 × 3 × 2 × 1 = 120

実行例

  print(factorial(0))   # 1
print(factorial(5))   # 120
print(factorial(10))  # 3628800

✏️ やること

  • ベースケース：n == 0 のとき 1 を返す
  • 再帰ケース：n * factorial(n - 1) を返す
  • if 文でベースケースと再帰ケースを分岐させる

▶ 💡 ヒント

  • まずベースケースを書きましょう：
    def factorial(n):
        if n == 0:
            return ???
    0の階乗は何でしたか？
  • 次に再帰ケースを書きます：
    def factorial(n):
        if n == 0:
            return 1
        return n * factorial(???)
    n! は n × (n-1)! なので、factorial に何を渡せばよいですか？
  • 完成した関数はこうなります：
    def factorial(n):
        if n == 0:
            return 1
        return n * factorial(n - 1)
