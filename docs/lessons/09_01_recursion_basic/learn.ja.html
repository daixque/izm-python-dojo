再帰の基礎

📚 説明
English

🔄 再帰（さいき）とは何か

再帰とは、関数が自分自身を呼び出すプログラミングの技法です。

最初は少し難しく聞こえるかもしれませんが、実は日常生活にも似たようなものがあります。

たとえば「入れ子になった箱を全部開けて中身を確認する」作業を考えてみましょう。箱を開けたらまた箱があって、それを開けたらまた箱があって…という繰り返しです。

再帰を使うと、このような「同じ構造が繰り返される」問題をシンプルなコードで解決できます。

再帰の基本イメージ
def count_down(n):
    if n == 0:
        print("発射！")
    else:
        print(n)
        count_down(n - 1)  # 自分自身を呼び出す！

count_down(3)
# 3
# 2
# 1
# 発射！
count_down 関数の中で count_down(n - 1) と自分自身を呼び出しています。
これが再帰です！

🛑 ベースケースと再帰ケース

再帰関数には必ず2つの部分が必要です。

  • ベースケース（終了条件）：これ以上再帰しない条件。再帰を止めるための「出口」です。
  • 再帰ケース：自分自身を呼び出す部分。問題を少しずつ小さくしていきます。

ベースケースがないと、関数が永遠に自分自身を呼び続ける「無限再帰」になってしまいます！

ベースケースと再帰ケースの構造
def count_down(n):
    # ベースケース（終了条件）
    if n == 0:
        print("発射！")
        return  # ここで終了！
    
    # 再帰ケース（問題を小さくして再帰）
    print(n)
    count_down(n - 1)  # n を 1 減らして呼び出す
ベースケース: n == 0 のとき → 終了
再帰ケース: n > 0 のとき → n - 1 で自分を呼ぶ

🔢 階乗（かいじょう）を再帰で計算する

まず、階乗とは何かを説明します。

階乗は n! と書き、1 から n までの整数を全部かけ合わせた数です。

  • 1! = 1（1だけ）
  • 3! = 3 × 2 × 1 = 6
  • 5! = 5 × 4 × 3 × 2 × 1 = 120

特別なルールとして、0! = 1 と定義されています（覚えておきましょう）。

階乗には面白いパターンがあります：n! = n × (n-1)!

たとえば 5! = 5 × 4!、4! = 4 × 3! … というように、「n の階乗は、n に (n-1) の階乗をかけたもの」と表せます。これはまさに再帰のパターンです！

階乗を再帰で実装する
def factorial(n):
    # ベースケース：0! = 1
    if n == 0:
        return 1
    
    # 再帰ケース：n! = n × (n-1)!
    return n * factorial(n - 1)

print(factorial(0))  # 1
print(factorial(3))  # 6
print(factorial(5))  # 120
factorial(5) を呼ぶと…
→ 5 * factorial(4)
→ 5 * 4 * factorial(3)
→ 5 * 4 * 3 * factorial(2)
→ 5 * 4 * 3 * 2 * factorial(1)
→ 5 * 4 * 3 * 2 * 1 * factorial(0)
→ 5 * 4 * 3 * 2 * 1 * 1 = 120

⚠️ 無限再帰に注意！

ベースケースを書き忘れると、関数が永遠に自分自身を呼び続けてしまいます。これを無限再帰と言います。

Pythonでは一定の深さを超えると自動的に RecursionError というエラーが発生して止まりますが、プログラムがクラッシュしてしまいます。

危険な例：ベースケースなし
# ❌ これはダメ！無限再帰になる
def bad_count(n):
    print(n)
    bad_count(n - 1)  # 止まらない！

# ✅ これが正しい
def good_count(n):
    if n == 0:  # ベースケースが必要
        return
    print(n)
    good_count(n - 1)
再帰関数を書くときは、必ずベースケースを最初に考えましょう。
「どうなったら再帰を止めるか？」を先に決めることが大切です。

💡 まとめ

  • 再帰とは、関数が自分自身を呼び出す技法
  • ベースケース（終了条件）が必ず必要 — ないと無限再帰になる
  • 再帰ケースで問題を少しずつ小さくして、ベースケースに近づける
  • 階乗は再帰の代表的な例：n! = n × (n-1)!、0! = 1
