🔗 What are Logical Operators?

In the previous lesson, we learned about if, elif, and else. Now we'll learn how to combine multiple conditions.

Logical operators allow you to express complex conditions like "A AND B", "A OR B", and "NOT A".

✅ and Operator - "AND"

and returns True only when both conditions are True.

Basic and
age = 25
height = 170

if age >= 18 and height >= 160:
    print("Conditions met")
else:
    print("Conditions not met")
In this example, "Conditions met" is printed only when age is 18 or more AND height is 160cm or more.
Truth Table for and
# Pattern 1
True and True # → True

# Pattern 2
True and False # → False

# Pattern 3
False and True # → False

# Pattern 4
False and False # → False
Important: and returns True only when both are True.

🔀 or Operator - "OR"

or returns True if either condition is True.

Basic or
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's a weekend")
else:
    print("It's a weekday")
In this example, "It's a weekend" is printed when the day is Saturday OR Sunday.
Truth Table for or
# Pattern 1
True or True # → True

# Pattern 2
True or False # → True

# Pattern 3
False or True # → True

# Pattern 4
False or False # → False
Important: or returns True if either one is True.

🚫 not Operator - "NOT"

not reverses the condition. It turns True into False and False into True.

Basic not
is_raining = False

if not is_raining:
    print("No umbrella needed")
Since is_raining is False, not is_raining becomes True, and "No umbrella needed" is printed.
Truth Table for not
not True # → False
not False # → True
Important: not flips True and False.

🔧 Complex Conditions

By combining logical operators, you can express more complex conditions.

Combining Multiple Logical Operators
age = 20
has_ticket = True
is_holiday = False

if (age >= 18 and has_ticket) or is_holiday:
    print("You can enter")
This condition reads as:
"(18 or older AND has a ticket) OR it's a holiday" allows entry.
Order of Evaluation
# With parentheses
if (age >= 18 and height >= 160) or has_special_pass:
    print("OK")

# Without parentheses
if age >= 18 and height >= 160 or has_special_pass:
    print("OK") # Same meaning
Priority order:
1. Expressions in parentheses have highest priority
2. not
3. and
4. or

When in doubt, use parentheses to be explicit!

📊 Practical Example - Theme Park Ride Requirements

Let's look at a practical example of logical operators: checking requirements for a theme park ride.

Checking Requirements
age = 10
height = 130

# Ride requirements: 6 years or older AND 120cm or taller
if age >= 6 and height >= 120:
    print("You can ride this attraction")
else:
    print("You cannot ride this attraction")
In this example, both age and height requirements must be met.
Since we're using and, failing either condition means you can't ride.

💡 Summary

  • and: True only when both are True (strict condition)
  • or: True when either is True (lenient condition)
  • not: Reverses True and False
  • Use parentheses () to make priority clear
