📋 Task

Implement the function normalize(data).

It receives a list of numbers data and returns the min-max normalized result.

What is Min-Max Normalization?

It rescales the data so the minimum value becomes 0 and the maximum value becomes 1.

Formula: (x - min) / (max - min)

Example Output

  print(normalize([0, 5, 10]))
# [0.0, 0.5, 1.0]

print(normalize([2, 4, 6, 8]))
# [0.0, 0.333..., 0.666..., 1.0]

✏️ Instructions

  • The function signature def normalize(data): is already provided
  • Use np.array(data, dtype=float) to convert the list to a float NumPy array
  • Use arr.min() and arr.max() to get the minimum and maximum values
  • Apply the formula (arr - arr.min()) / (arr.max() - arr.min()) and return the result
