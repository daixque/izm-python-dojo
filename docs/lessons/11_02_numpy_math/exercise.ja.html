📋 課題内容

関数 normalize(data) を実装しましょう。

数値のリスト data を受け取り、min-max正規化した結果を返します。

min-max正規化とは

データの最小値を0、最大値を1になるよう変換する方法です。

計算式: (x - 最小値) / (最大値 - 最小値)

実行例

print(normalize([0, 5, 10]))
# [0.0, 0.5, 1.0]

print(normalize([2, 4, 6, 8]))
# [0.0, 0.333..., 0.666..., 1.0]

✏️ やること

  • 関数シグネチャ def normalize(data): は用意済みです
