📋 Task

Implement the function array_stats(data).

It receives a list of numbers data and returns a dictionary containing the sum and mean.

Specification

  • Convert the argument data into a NumPy array
  • Compute the sum and mean
  • Return them as {"sum": float(total), "mean": float(average)}

Example Output

  print(array_stats([1, 2, 3, 4, 5]))
# {"sum": 15.0, "mean": 3.0}

print(array_stats([10, 20, 30]))
# {"sum": 60.0, "mean": 20.0}

✏️ Instructions

  • The function signature def array_stats(data): is already provided
  • Use np.array(data) to convert the list into a NumPy array
  • Use arr.sum() for the sum and arr.mean() for the mean
  • Wrap results with float() to return standard Python floats

▶ 💡 Hints

  • First convert to a NumPy array:
    arr = np.array(data)
  • Use the sum() and mean() methods:
    arr.sum() ← sum
    arr.mean() ← mean
  • Build and return a dictionary:
    return {"sum": float(arr.sum()), "mean": float(arr.mean())}
