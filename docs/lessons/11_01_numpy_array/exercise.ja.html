📋 課題内容

関数 array_stats(data) を実装しましょう。

数値のリスト data を受け取り、合計（sum）と平均（mean）を辞書で返します。

仕様

  • 引数 data をNumPy配列に変換する
  • 合計と平均を計算する
  • {"sum": float(合計), "mean": float(平均)} の形式で返す

実行例

print(array_stats([1, 2, 3, 4, 5]))
# {"sum": 15.0, "mean": 3.0}

print(array_stats([10, 20, 30]))
# {"sum": 60.0, "mean": 20.0}

✏️ やること

  • 関数シグネチャ def array_stats(data): は用意済みです
  • np.array(data) でリストをNumPy配列に変換します
  • arr.sum() で合計、arr.mean() で平均を計算できます
  • float() で数値をPythonのfloat型に変換して返します

▶ 💡 ヒント

  • まず NumPy 配列に変換します：
    arr = np.array(data)
  • sum() と mean() メソッドを使います：
    arr.sum() ← 合計
    arr.mean() ← 平均
  • 辞書を作って返します：
    return {"sum": float(arr.sum()), "mean": float(arr.mean())}
