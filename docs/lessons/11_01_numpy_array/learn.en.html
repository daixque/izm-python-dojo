📦 What is NumPy?

NumPy is a Python library for fast numerical computation.

While Python's built-in lists can do calculations, NumPy is many times faster.

It is used in science, math, data analysis, artificial intelligence, and many other fields.

To use NumPy, start your program with import numpy as np.

Importing NumPy
import numpy as np

# Now we can use NumPy features with the name "np"
as np means "use NumPy under the short name np".
This convention is used universally by NumPy programmers worldwide.

🔢 Creating Arrays with np.array()

NumPy arrays (ndarray) are created with np.array().

Pass a Python list and it will be converted into a NumPy array.

Creating an Array
import numpy as np

a = np.array([1, 2, 3, 4, 5])
print(a) # [1 2 3 4 5]
print(type(a)) # <class 'numpy.ndarray'>
Passing a list to np.array() creates a NumPy array.
The type of a NumPy array is called ndarray.
Array of Floats
import numpy as np

b = np.array([1.5, 2.5, 3.5])
print(b) # [1.5 2.5 3.5]
print(b.dtype) # float64
When you include decimal numbers, the array automatically uses the floating-point type (float64).

📐 shape and dtype

NumPy arrays have two important attributes: shape (dimensions) and dtype (data type).

Checking shape and dtype
import numpy as np

a = np.array([10, 20, 30, 40])
print(a.shape) # (4,) ← 1D array with 4 elements
print(a.dtype) # int64 ← integer type
shape describes the "form" of the array. (4,) means a 1-dimensional array with 4 elements.
dtype shows the type of data stored (integers, floats, etc.).

✂️ Indexing and Slicing

Accessing elements in a NumPy array works the same way as Python lists.

Accessing Elements by Index
import numpy as np

a = np.array([10, 20, 30, 40, 50])
print(a[0]) # 10 (first element)
print(a[2]) # 30 (third element)
print(a[-1]) # 50 (last element)
Indices start at 0.
Negative indices count from the end of the array.
Slicing to Get a Sub-array
import numpy as np

a = np.array([10, 20, 30, 40, 50])
print(a[1:4]) # [20 30 40]
print(a[:3]) # [10 20 30]
print(a[2:]) # [30 40 50]
Use a[start:stop] to extract a portion of the array.
The element at the stop index is not included (same rule as Python lists).

➕ Array Arithmetic

NumPy array operations are performed element-wise.

This is very different from Python lists.

Operations Between Arrays
import numpy as np

a = np.array([1, 2, 3, 4])
b = np.array([10, 20, 30, 40])

print(a + b) # [11 22 33 44]
print(b - a) # [ 9 18 27 36]
print(a * b) # [ 10 40 90 160]
print(b / a) # [10. 10. 10. 10.]
Each element is combined with the corresponding element at the same position.
a + b is equivalent to [1+10, 2+20, 3+30, 4+40].
Operations with a Scalar
import numpy as np

a = np.array([1, 2, 3, 4, 5])

print(a * 2) # [ 2 4 6 8 10]
print(a + 100) # [101 102 103 104 105]
print(a ** 2) # [ 1 4 9 16 25]
When combining an array with a single number (scalar), the operation is applied to every element.
This is called broadcasting.

💡 Summary

  • Import NumPy with import numpy as np
  • Create a NumPy array from a list using np.array([...])
  • Use .shape to check dimensions and .dtype to check the data type
  • Indexing and slicing work the same way as Python lists
  • +, -, *, / between arrays operate element-wise
