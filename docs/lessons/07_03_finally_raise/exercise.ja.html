📋 課題内容

関数 validate_age(age) を実装しましょう。

age が 0 以上 150 以下であれば age をそのまま返します。

範囲外の場合は ValueError を raise します。

実行例

  print(validate_age(25))   # 25
validate_age(-1)          # ValueError が発生
validate_age(200)         # ValueError が発生

✏️ やること

  • 関数シグネチャ def validate_age(age): は用意済みです
  • age < 0 または age > 150 の場合に ValueError を raise します
  • 正常な場合は age をそのまま return します
  • エラーメッセージは自由に設定してください

▶ 💡 ヒント

  • 条件によって ValueError を raise します：
    def validate_age(age):
        if age < 0 or age > 150:
            raise ValueError("...")
        return age
  • raise の書き方：
    raise ValueError("年齢が無効です")
    エラーメッセージには、何が問題かを書くといいです。
  • 2つの条件を or でつなげます：
    if age < 0 or age > 150:
