📋 Task

Implement the function validate_age(age).

If age is between 0 and 150 (inclusive), return age as-is.

If it is out of range, raise a ValueError.

Example Output

  print(validate_age(25))   # 25
validate_age(-1)          # raises ValueError
validate_age(200)         # raises ValueError

✏️ Instructions

  • The function signature def validate_age(age): is already provided
  • Raise a ValueError if age < 0 or age > 150
  • Return age as-is if it is a valid value
  • You can set any error message you like

▶ 💡 Hints

  • Raise a ValueError based on the condition:
    def validate_age(age):
        if age < 0 or age > 150:
            raise ValueError("...")
        return age
  • How to use raise:
    raise ValueError("Invalid age")
    A good error message explains what went wrong.
  • Connect two conditions with or:
    if age < 0 or age > 150:
