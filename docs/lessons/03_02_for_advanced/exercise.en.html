📋 Task

Implement the function sum_numbers(n).

This function returns the sum of all integers from 1 to n.

Example Output

For example, calling sum_numbers(5) computes:

  1 + 2 + 3 + 4 + 5 = 15

so it returns 15.

sum_numbers(10) returns 55, and sum_numbers(100) returns 5050.

✏️ Instructions

  • The function definition and the initialization total = 0 are already provided. Implement the loop.
  • Use a for loop with range(1, n + 1) to iterate from 1 to n
  • Add each value to the total each iteration using total += i
  • After the loop, return the total with return total (already written for you)

▶ 💡 Hints

  • Pattern for computing a sum:
    total = 0
    for i in range(1, n + 1):
        total += i
  • range(1, n + 1) generates 1, 2, 3, ..., n.
    We write n + 1 because the stop value is excluded.
  • total += i is the same as total = total + i.
    Each iteration adds the current value of i to total.
  • The completed function looks like this:
    def sum_numbers(n):
        total = 0
        for i in range(1, n + 1):
            total += i
        return total
