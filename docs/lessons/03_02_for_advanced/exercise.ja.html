📋 課題内容

関数 sum_numbers(n) を実装しましょう。

この関数は 1 から n までの整数の合計を返します。

出力例

例えば sum_numbers(5) を呼び出すと：

  1 + 2 + 3 + 4 + 5 = 15

なので 15 を返します。

sum_numbers(10) は 55 を、sum_numbers(100) は 5050 を返します。

✏️ やること

  • 関数の定義と total = 0 の初期化は用意済みです。ループ部分を実装してください
  • for ループと range(1, n + 1) を使って 1 から n まで繰り返します
  • ループのたびに total += i で合計に加算します
  • ループが終わったら return total で合計を返します（return total はすでに書かれています）

▶ 💡 ヒント

  • 合計を求めるパターン：
    total = 0
    for i in range(1, n + 1):
        total += i
  • range(1, n + 1) は 1, 2, 3, ..., n を生成します。
    n + 1 としているのは、終了値が含まれないためです。
  • total += i は total = total + i と同じ意味です。
    ループのたびに i の値が total に加算されていきます。
  • 完成した関数はこのような形になります：
    def sum_numbers(n):
        total = 0
        for i in range(1, n + 1):
            total += i
        return total
