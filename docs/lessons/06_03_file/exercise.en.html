📋 Task

As a basic feature of a memo app, implement two functions to save and load text files.

  1. save_memo(filename, content): A function that saves text to a file
  2. load_memo(filename): A function that reads text from a file

✏️ Instructions

  • In the save_memo function, use with open(filename, 'w', encoding='utf-8') as f: to open the file, and f.write(content) to write the content
  • In the load_memo function, use with open(filename, 'r', encoding='utf-8') as f: inside a try: block, and return the content with f.read()
  • If the file does not exist (FileNotFoundError), return None
  • Do not modify the test code outside the functions

▶ 💡 Hints

  • Basic structure of the save_memo function:
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(content)
  • Basic structure of the load_memo function:
    try:
        with open(filename, 'r', encoding='utf-8') as f:
            return f.read()
    except FileNotFoundError:
        return None
  • pass means "do nothing". Once you've written your implementation, remove the pass.
  • To handle missing files: calling load_memo("notfound.txt") raises FileNotFoundError. Catch it with except FileNotFoundError: and return None.
