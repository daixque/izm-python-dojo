📋 課題内容

メモ帳アプリの基本機能として、テキストファイルを保存・読み込みする2つの関数を実装しましょう。

  1. save_memo(filename, content)：テキストをファイルに保存する関数
  2. load_memo(filename)：ファイルからテキストを読み込む関数

✏️ やること

  • save_memo 関数では with open(filename, 'w', encoding='utf-8') as f: を使ってファイルを開き、f.write(content) で内容を書き込みます
  • load_memo 関数では try: ブロック内で with open(filename, 'r', encoding='utf-8') as f: を使ってファイルを開き、f.read() で内容を返します
  • ファイルが存在しない場合（FileNotFoundError）は None を返してください
  • 関数の外のテストコードは変更しないでください

▶ 💡 ヒント

  • save_memo 関数の基本構造：
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(content)
  • load_memo 関数の基本構造：
    try:
        with open(filename, 'r', encoding='utf-8') as f:
            return f.read()
    except FileNotFoundError:
        return None
  • pass は「何もしない」という意味です。実装を書いたら pass は削除してください。
  • ファイルが存在しない場合の確認：load_memo("notfound.txt") を呼ぶと FileNotFoundError が発生します。except FileNotFoundError: でキャッチして return None を返しましょう。
