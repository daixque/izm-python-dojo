📋 Task

Write a program that converts two numbers given as strings to integers, calculates their sum, and displays it.

Finally, display the result in the format "The sum is ○○".

✏️ Instructions

  • Assign the string "50" to variable num1
  • Assign the string "30" to variable num2
  • Use int() to convert both to integers and calculate the sum
  • Display the result in the format 'The sum is ○○'

▶ 💡 Hints

  • You can convert a string to integer with int(num1)
  • Calculate the sum with int(num1) + int(num2)
  • Use str() to convert numbers to strings
  • Example: print("The sum is " + str(sum))
