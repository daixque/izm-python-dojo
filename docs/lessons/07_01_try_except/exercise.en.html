📋 Task

Implement the function safe_int(value).

It converts the string value to an integer and returns it. If the conversion fails (a ValueError is raised), return None.

Example Output

print(safe_int("42"))    # 42
print(safe_int("abc"))   # None
print(safe_int("3.14"))  # None

✏️ Instructions

  • The function signature def safe_int(value): is already provided
  • Use int(value) inside a try: block to attempt the conversion
  • Catch the error using except ValueError:
  • Return the converted integer if the conversion succeeds
  • Return None if the conversion fails

▶ 💡 Hints

  • Basic try/except structure:
    def safe_int(value):
        try:
            return int(value)
        except ValueError:
            return ???
  • Return None when the error occurs:
    except ValueError:
        return None
  • Strings like "3.14" that contain a decimal point cannot be converted by int(), so they raise a ValueError.
    float("3.14") succeeds, but int("3.14") raises an error.
