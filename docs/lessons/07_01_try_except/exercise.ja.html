📋 課題内容

関数 safe_int(value) を実装しましょう。

文字列 value を整数に変換して返します。変換できない場合（ValueError が発生した場合）は None を返します。

実行例

  print(safe_int("42"))    # 42
print(safe_int("abc"))   # None
print(safe_int("3.14"))  # None

✏️ やること

  • 関数シグネチャ def safe_int(value): は用意済みです
  • try: ブロックの中で int(value) を使って変換します
  • except ValueError: でエラーをキャッチします
  • 変換に成功したら変換後の整数を return で返します
  • 変換に失敗したら None を return で返します

▶ 💡 ヒント

  • try/except の基本的な形：
    def safe_int(value):
        try:
            return int(value)
        except ValueError:
            return ???
  • エラーが発生したときは None を返します：
    except ValueError:
        return None
  • "3.14" のような小数点を含む文字列は int() で変換できないため、ValueError になります。
    float("3.14") は成功しますが、int("3.14") はエラーになります。
