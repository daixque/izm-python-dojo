📋 課題内容

関数 fibonacci(n) を再帰を使って実装しましょう。

フィボナッチ数列のn番目の値を返します。

  • fibonacci(0) → 0
  • fibonacci(1) → 1
  • fibonacci(n) → fibonacci(n-1) + fibonacci(n-2)（n ≥ 2 のとき）

実行例

print(fibonacci(0))   # 0
print(fibonacci(1))   # 1
print(fibonacci(6))   # 8
print(fibonacci(10))  # 55

✏️ やること

  • ベースケース1：n == 0 のとき 0 を返す
  • ベースケース2：n == 1 のとき 1 を返す
  • 再帰ケース：fibonacci(n-1) + fibonacci(n-2) を返す

▶ 💡 ヒント

  • ベースケースは2つあります：
    def fibonacci(n):
        if n == 0:
            return ???
        if n == 1:
            return ???
  • 再帰ケースでは2つの再帰呼び出しを足します：
    return fibonacci(n - 1) + fibonacci(???)
  • 完成した関数はこうなります：
    def fibonacci(n):
        if n == 0:
            return 0
        if n == 1:
            return 1
        return fibonacci(n - 1) + fibonacci(n - 2)
