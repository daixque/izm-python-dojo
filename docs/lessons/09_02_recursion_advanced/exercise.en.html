📋 Task

Implement the function fibonacci(n) using recursion.

It should return the nth value in the Fibonacci sequence.

  • fibonacci(0) → 0
  • fibonacci(1) → 1
  • fibonacci(n) → fibonacci(n-1) + fibonacci(n-2) (when n ≥ 2)

Example Output

  print(fibonacci(0))   # 0
print(fibonacci(1))   # 1
print(fibonacci(6))   # 8
print(fibonacci(10))  # 55

✏️ Instructions

  • Base case 1: when n == 0, return 0
  • Base case 2: when n == 1, return 1
  • Recursive case: return fibonacci(n-1) + fibonacci(n-2)

▶ 💡 Hints

  • There are two base cases:
    def fibonacci(n):
        if n == 0:
            return ???
        if n == 1:
            return ???
  • In the recursive case, add two recursive calls together:
    return fibonacci(n - 1) + fibonacci(???)
  • The completed function looks like this:
    def fibonacci(n):
        if n == 0:
            return 0
        if n == 1:
            return 1
        return fibonacci(n - 1) + fibonacci(n - 2)
