📋 Task

Implement the function get_score(students, name).

students is a dictionary with student names as keys and scores as values. Return the score for the student specified by name.

If the student does not exist in the dictionary, return -1.

Examples

  students = {"Alice": 85, "Bob": 92, "Charlie": 78}
get_score(students, "Alice")    # 85
get_score(students, "Bob")      # 92
get_score(students, "Dave")     # -1 (not found)

✏️ Instructions

  • Use the in operator to check whether name exists in students
  • If it exists, return students[name]
  • If it does not exist, return -1

▶ 💡 Hints

  • Use the in operator to check for the key:
    if name in students:
        # name exists in the dictionary
  • If it exists, return the value from the dictionary:
    return students[name]
  • If it does not exist, return -1:
    else:
        return -1
  • You can also write this in one line using .get():
    return students.get(name, -1)
