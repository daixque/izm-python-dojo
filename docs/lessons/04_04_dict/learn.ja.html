辞書の基礎

📚 説明
English

📖 辞書とは

辞書（dict）は、キーと値のペアでデータを管理するデータ型です。

現実の辞書で「単語 → 意味」を引くように、キーを使って素早く値を取り出せます。

例えば、生徒の名前と点数を対応させたり、単語と翻訳を対応させたりするときに便利です。

辞書の作成
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
print(scores)
{キー: 値, キー: 値, ...} の形で作ります。
キーと値は : で区切り、各ペアは , で区切ります。
キーには文字列や数値、値には任意の型が使えます。
値へのアクセス
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
print(scores["Alice"]) # 85
print(scores["Bob"]) # 92
辞書名[キー] の形でキーに対応する値を取得できます。
存在しないキーを指定すると KeyError が発生します。

🔍 in 演算子による存在確認

キーが辞書に存在するかどうかは in 演算子で確認できます。

in 演算子の使い方
scores = {"Alice": 85, "Bob": 92}
print("Alice" in scores) # True
print("Dave" in scores) # False

if "Alice" in scores:
    print(f"Alice の点数: {scores['Alice']}")
キー in 辞書 は、キーが辞書に存在すれば True、なければ False を返します。
if と組み合わせて、安全に値を取得できます。

✏️ 要素の追加・変更・削除

辞書は作成後に要素を追加・変更・削除できます。

要素の追加と変更
scores = {"Alice": 85, "Bob": 92}

# 新しいキーに代入すると追加
scores["Dave"] = 75
print(scores) # {"Alice": 85, "Bob": 92, "Dave": 75}

# 既存のキーに代入すると変更
scores["Alice"] = 90
print(scores["Alice"]) # 90
辞書[キー] = 値 で追加・変更ができます。
キーが存在しなければ追加、存在すれば値が上書きされます。
要素の削除
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
del scores["Charlie"]
print(scores) # {"Alice": 85, "Bob": 92}
del 辞書[キー] でキーと値のペアを削除できます。

🔑 keys(), values(), items() メソッド

辞書の全キー・全値・全ペアを取得するメソッドがあります。

keys(), values(), items()
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}

print(list(scores.keys())) # ["Alice", "Bob", "Charlie"]
print(list(scores.values())) # [85, 92, 78]
print(list(scores.items())) # [("Alice", 85), ("Bob", 92), ...]
keys(): 全キーを返す
values(): 全値を返す
items(): (キー, 値) のタプルを返す
for ループで辞書を走査する
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
for name, score in scores.items():
    print(f"{name}: {score}点")
# 出力：
# Alice: 85点
# Bob: 92点
# Charlie: 78点
for キー, 値 in 辞書.items(): で全ペアをループ処理できます。
全生徒の点数を表示したり、集計したりするときに使います。

🛡️ .get() メソッドで安全に値を取得する

存在しないキーにアクセスしても KeyError が出ないようにするには .get() を使います。

.get() メソッドの使い方
scores = {"Alice": 85, "Bob": 92}

# キーが存在する場合
print(scores.get("Alice")) # 85

# キーが存在しない場合（デフォルト値なし）
print(scores.get("Dave")) # None

# キーが存在しない場合（デフォルト値あり）
print(scores.get("Dave", 0)) # 0
辞書.get(キー): キーがなければ None を返す
辞書.get(キー, デフォルト値): キーがなければデフォルト値を返す
KeyError が発生しないので安全です。

💡 まとめ

  • 辞書は {キー: 値} の形でデータを管理する
  • 辞書[キー] で値を取得できる（存在しないと KeyError）
  • キー in 辞書 でキーの存在を確認できる
  • 辞書[キー] = 値 で追加・変更ができる
  • items() で全ペアを for ループで処理できる
  • 辞書.get(キー, デフォルト) で安全に値を取得できる
