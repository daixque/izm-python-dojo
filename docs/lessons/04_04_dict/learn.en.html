📖 What is a Dictionary?

A dictionary (dict) is a data type that stores data as key-value pairs.

Just like looking up a word in a real dictionary to find its meaning, you can use a key to quickly retrieve a value.

Dictionaries are useful for things like mapping student names to scores, or words to translations.

Creating a Dictionary
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
print(scores)
Create a dictionary using the {key: value, key: value, ...} format.
Separate each key and value with :, and separate each pair with ,.
Keys are typically strings or numbers; values can be any type.
Accessing Values
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
print(scores["Alice"]) # 85
print(scores["Bob"]) # 92
Use dictionary[key] to retrieve the value for that key.
If you specify a key that doesn't exist, a KeyError will occur.

🔍 Checking for Keys with the in Operator

Use the in operator to check whether a key exists in a dictionary.

Using the in Operator
scores = {"Alice": 85, "Bob": 92}
print("Alice" in scores) # True
print("Dave" in scores) # False

if "Alice" in scores:
    print(f"Alice's score: {scores['Alice']}")
key in dictionary returns True if the key exists, and False otherwise.
Combining it with if lets you retrieve values safely.

✏️ Adding, Changing, and Removing Elements

You can add, change, and remove elements from a dictionary after it is created.

Adding and Changing Elements
scores = {"Alice": 85, "Bob": 92}

# Assigning to a new key adds it
scores["Dave"] = 75
print(scores) # {"Alice": 85, "Bob": 92, "Dave": 75}

# Assigning to an existing key changes its value
scores["Alice"] = 90
print(scores["Alice"]) # 90
Use dictionary[key] = value to add or change entries.
If the key doesn't exist it is added; if it already exists its value is overwritten.
Removing Elements
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
del scores["Charlie"]
print(scores) # {"Alice": 85, "Bob": 92}
Use del dictionary[key] to remove a key-value pair.

🔑 keys(), values(), and items() Methods

There are methods to retrieve all keys, all values, or all pairs from a dictionary.

keys(), values(), items()
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}

print(list(scores.keys())) # ["Alice", "Bob", "Charlie"]
print(list(scores.values())) # [85, 92, 78]
print(list(scores.items())) # [("Alice", 85), ("Bob", 92), ...]
keys(): returns all keys
values(): returns all values
items(): returns (key, value) tuples
Iterating Over a Dictionary with a for Loop
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
for name, score in scores.items():
    print(f"{name}: {score} points")
# Output:
# Alice: 85 points
# Bob: 92 points
# Charlie: 78 points
Use for key, value in dictionary.items(): to loop over all pairs.
This is great for displaying or summarizing all entries.

🛡️ Safe Value Access with .get()

To avoid KeyError when a key might not exist, use the .get() method.

Using .get()
scores = {"Alice": 85, "Bob": 92}

# Key exists
print(scores.get("Alice")) # 85

# Key does not exist (no default)
print(scores.get("Dave")) # None

# Key does not exist (with default value)
print(scores.get("Dave", 0)) # 0
dictionary.get(key): returns None if the key doesn't exist
dictionary.get(key, default): returns the default value if the key doesn't exist
No KeyError is raised, making it much safer.

💡 Summary

  • A dictionary stores data as {key: value} pairs
  • dictionary[key] retrieves a value (raises KeyError if key is missing)
  • key in dictionary checks whether a key exists
  • dictionary[key] = value adds or updates an entry
  • items() lets you loop over all key-value pairs
  • dictionary.get(key, default) safely retrieves a value
