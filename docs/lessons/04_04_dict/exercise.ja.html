📋 課題内容

関数 get_score(students, name) を実装しましょう。

students は生徒名をキー、点数を値とする辞書です。name で指定された生徒の点数を返します。

生徒が辞書に存在しない場合は -1 を返してください。

実行例

students = {"Alice": 85, "Bob": 92, "Charlie": 78}
get_score(students, "Alice")    # 85
get_score(students, "Bob")      # 92
get_score(students, "Dave")     # -1（存在しない）

✏️ やること

  • in 演算子を使って name が students に存在するか確認します
  • 存在する場合は students[name] を返します
  • 存在しない場合は -1 を返します

▶ 💡 ヒント

  • in 演算子でキーの存在を確認します：
    if name in students:
        # name が辞書に存在する
  • 存在する場合は辞書からその値を返します：
    return students[name]
  • 存在しない場合は -1 を返します：
    else:
        return -1
  • .get() を使って1行で書くこともできます：
    return students.get(name, -1)
