📋 課題内容

関数 bubble_sort(numbers) をバブルソートで実装しましょう。

数値のリスト numbers を昇順（小さい順）に並び替えた新しいリストを返します。

注意：sort() や sorted() は使わないで実装してください！

実行例

  print(bubble_sort([5, 3, 8, 1, 2]))     # [1, 2, 3, 5, 8]
print(bubble_sort([-3, 0, -1, 2]))      # [-3, -1, 0, 2]
print(bubble_sort([1]))                 # [1]

✏️ やること

  • numbers.copy() でコピーを作り、元のリストを変更しないようにする
  • 外側のループで n 回パスする：for i in range(n):
  • 内側のループで隣り合う要素を比べる：for j in range(n - 1 - i):
  • result[j] > result[j+1] なら result[j], result[j+1] = result[j+1], result[j] で交換
  • 最後に result を返す

▶ 💡 ヒント

  • まずコピーを作ってリストの長さを取得します：
    def bubble_sort(numbers):
        result = numbers.copy()
        n = len(result)
        for i in range(n):
            for j in range(n - 1 - i):
                # ここに比較と交換を書く
  • 隣り合う要素を比べて、逆順なら交換します：
    if result[j] > result[j + 1]:
        result[j], result[j + 1] = result[j + 1], result[j]
  • 完成した関数はこうなります：
    def bubble_sort(numbers):
        result = numbers.copy()
        n = len(result)
        for i in range(n):
            for j in range(n - 1 - i):
                if result[j] > result[j + 1]:
                    result[j], result[j + 1] = result[j + 1], result[j]
        return result
