English

🔢 ソートとは何か

ソート（sort）とは、データを特定の順番（昇順・降順など）に並び替えることです。

たとえば成績の高い順に並べたり、名前をあいうえお順に並べたりするのがソートです。

コンピュータプログラムでもデータを並び替えることは非常に多く、ソートアルゴリズムはプログラミングの基本として重要です。

Pythonには sorted() という便利な組み込み関数がありますが、今回はソートの仕組みを理解するために、自分でアルゴリズムを実装してみましょう。

🫧 バブルソート（Bubble Sort）

バブルソートは、隣り合う2つの要素を比べて、順番が逆なら交換する、という操作を繰り返すソートアルゴリズムです。

大きな数が泡（バブル）のようにリストの右端へ浮かび上がっていくことから「バブルソート」と呼ばれます。

仕組みを詳しく説明します

リスト [5, 3, 8, 1] を昇順（小さい順）に並べる例で見てみましょう。

1回目のパス（先頭から）：

  • 5 と 3 を比較 → 5 > 3 なので交換 → [3, 5, 8, 1]
  • 5 と 8 を比較 → 5 < 8 なのでそのまま → [3, 5, 8, 1]
  • 8 と 1 を比較 → 8 > 1 なので交換 → [3, 5, 1, 8]
  • 最大値 8 が右端に来た！

2回目のパス：

  • 3 と 5 を比較 → そのまま → [3, 5, 1, 8]
  • 5 と 1 を比較 → 交換 → [3, 1, 5, 8]

3回目のパス：

  • 3 と 1 を比較 → 交換 → [1, 3, 5, 8]

完成！[1, 3, 5, 8]

バブルソートの実装
def bubble_sort(numbers):
    # リストのコピーを作る（元のリストを変更しない）
    result = numbers.copy()
    n = len(result)
    
    for i in range(n):
        for j in range(n - 1 - i):
            if result[j] > result[j + 1]:
                # 隣同士を交換
                result[j], result[j + 1] = result[j + 1], result[j]
    
    return result

print(bubble_sort([5, 3, 8, 1, 2]))  # [1, 2, 3, 5, 8]
外側のループ: 何回パスするか（n回）
内側のループ: 隣り合う要素を比べて交換
n - 1 - i は、パスするたびに右端の確定済み要素を除外するため
Pythonの2つの変数の交換
# Pythonでは1行で2つの変数を交換できる
a = 5
b = 3
a, b = b, a  # 交換！
print(a)  # 3
print(b)  # 5
a, b = b, a で2つの変数の値を一度に交換できます。
他の言語では一時変数が必要なことが多いですが、Pythonではこう書けます。

🎯 選択ソート（Selection Sort）

選択ソートは、未整列部分から最小値を選んで先頭に移動させる操作を繰り返すアルゴリズムです。

リスト [5, 3, 8, 1] の例：

  • 全体から最小値 1 を選んで先頭と交換 → [1, 3, 8, 5]
  • 残りの [3, 8, 5] から最小値 3 を選ぶ → そのまま → [1, 3, 8, 5]
  • 残りの [8, 5] から最小値 5 を選んで交換 → [1, 3, 5, 8]
  • 完成！
選択ソートの実装
def selection_sort(numbers):
    result = numbers.copy()
    n = len(result)
    
    for i in range(n):
        # 未整列部分の最小値のインデックスを探す
        min_index = i
        for j in range(i + 1, n):
            if result[j] < result[min_index]:
                min_index = j
        # 最小値を先頭に移動
        result[i], result[min_index] = result[min_index], result[i]
    
    return result

print(selection_sort([5, 3, 8, 1, 2]))  # [1, 2, 3, 5, 8]
選択ソートは交換回数が少ないのが特徴です。
バブルソートと選択ソート、どちらも計算量は O(n²) です。

⚡ Python の組み込みソートと比較

Pythonには高性能な組み込みソートがあります。実際の開発ではこちらを使います。

sorted() 関数と sort() メソッド
numbers = [5, 3, 8, 1, 2]

# sorted() は新しいリストを返す
sorted_list = sorted(numbers)
print(sorted_list)  # [1, 2, 3, 5, 8]
print(numbers)      # [5, 3, 8, 1, 2] 元のリストは変わらない

# sort() はリストを直接並び替える（破壊的）
numbers.sort()
print(numbers)      # [1, 2, 3, 5, 8]
Pythonの sorted() は「ティムソート」という非常に高速なアルゴリズムを使っています。
計算量は O(n log n) で、バブルソートの O(n²) よりはるかに速いです！

💡 まとめ

  • バブルソート：隣り合う要素を比べて交換を繰り返す。実装が簡単。O(n²)。
  • 選択ソート：未整列部分から最小値を選んで先頭へ移動。O(n²)。
  • sorted() / sort()：Pythonの組み込みソート。O(n log n) でとても速い。
  • 実際の開発では sorted() を使う。自分で実装するのは仕組みを理解するため！
