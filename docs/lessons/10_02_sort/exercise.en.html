📋 Task

Implement the function bubble_sort(numbers) using the bubble sort algorithm.

Return a new list with the numbers from numbers sorted in ascending order (smallest first).

Important: Do not use sort() or sorted()!

Example Output

  print(bubble_sort([5, 3, 8, 1, 2]))     # [1, 2, 3, 5, 8]
print(bubble_sort([-3, 0, -1, 2]))      # [-3, -1, 0, 2]
print(bubble_sort([1]))                 # [1]

✏️ Instructions

  • Use numbers.copy() to create a copy so the original list is not modified
  • Outer loop runs n passes: for i in range(n):
  • Inner loop compares neighbors: for j in range(n - 1 - i):
  • If result[j] > result[j+1], swap: result[j], result[j+1] = result[j+1], result[j]
  • Return result at the end

    def bubble_sort(numbers):
        result = numbers.copy()
        n = len(result)
        for i in range(n):
            for j in range(n - 1 - i):
                if result[j] > result[j + 1]:
                    result[j], result[j + 1] = result[j + 1], result[j]
        return result
