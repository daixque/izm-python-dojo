🔢 What is Sorting?

Sorting means rearranging data in a specific order — such as ascending (smallest first) or descending (largest first).

Examples include ranking scores from highest to lowest, or putting names in alphabetical order.

Sorting is one of the most common operations in computer programs, so understanding sort algorithms is a fundamental programming skill.

Python has a built-in sorted() function, but in this lesson we'll implement sorting ourselves to understand how it works.

🫧 Bubble Sort

Bubble sort repeatedly compares two neighboring elements and swaps them if they are in the wrong order.

The name comes from the fact that large numbers "bubble up" to the right end of the list, just like bubbles rising in water.

Step-by-step walkthrough

Let's sort [5, 3, 8, 1] in ascending order (smallest first).

Pass 1 (from the beginning):

  • Compare 5 and 3 → 5 > 3, swap → [3, 5, 8, 1]
  • Compare 5 and 8 → 5 < 8, no swap → [3, 5, 8, 1]
  • Compare 8 and 1 → 8 > 1, swap → [3, 5, 1, 8]
  • The largest value 8 is now at the right end!

Pass 2:

  • Compare 3 and 5 → no swap → [3, 5, 1, 8]
  • Compare 5 and 1 → swap → [3, 1, 5, 8]

Pass 3:

  • Compare 3 and 1 → swap → [1, 3, 5, 8]

Done! [1, 3, 5, 8]

Bubble Sort Implementation
def bubble_sort(numbers):
    # Make a copy (don't modify the original)
    result = numbers.copy()
    n = len(result)
    
    for i in range(n):
        for j in range(n - 1 - i):
            if result[j] > result[j + 1]:
                # Swap neighbors
                result[j], result[j + 1] = result[j + 1], result[j]
    
    return result

print(bubble_sort([5, 3, 8, 1, 2]))  # [1, 2, 3, 5, 8]
Outer loop: how many passes to make (n times)
Inner loop: compare and swap neighboring elements
n - 1 - i excludes already-sorted elements at the right end
Swapping Two Variables in Python
# Python can swap two variables in one line
a = 5
b = 3
a, b = b, a  # swap!
print(a)  # 3
print(b)  # 5
a, b = b, a swaps two variables at once.
In many languages you need a temporary variable for this, but Python makes it simple!

🎯 Selection Sort

Selection sort repeatedly finds the minimum value from the unsorted portion and moves it to the front.

Example with [5, 3, 8, 1]:

  • Find minimum 1, swap with front → [1, 3, 8, 5]
  • Find minimum in [3, 8, 5] → 3 is already at front → [1, 3, 8, 5]
  • Find minimum in [8, 5] → 5, swap → [1, 3, 5, 8]
  • Done!
Selection Sort Implementation
def selection_sort(numbers):
    result = numbers.copy()
    n = len(result)
    
    for i in range(n):
        # Find index of minimum in unsorted portion
        min_index = i
        for j in range(i + 1, n):
            if result[j] < result[min_index]:
                min_index = j
        # Move minimum to the front
        result[i], result[min_index] = result[min_index], result[i]
    
    return result

print(selection_sort([5, 3, 8, 1, 2]))  # [1, 2, 3, 5, 8]
Selection sort makes fewer swaps than bubble sort.
Both bubble sort and selection sort are O(n²).

⚡ Comparing with Python's Built-in Sort

Python has a highly optimized built-in sort. In real projects, always use this.

sorted() Function and sort() Method
numbers = [5, 3, 8, 1, 2]

# sorted() returns a new list
sorted_list = sorted(numbers)
print(sorted_list)  # [1, 2, 3, 5, 8]
print(numbers)      # [5, 3, 8, 1, 2] original unchanged

# sort() modifies the list in place
numbers.sort()
print(numbers)      # [1, 2, 3, 5, 8]
Python's sorted() uses an algorithm called "Timsort" which is extremely fast.
Its complexity is O(n log n) — far faster than bubble sort's O(n²)!

💡 Summary

  • Bubble sort: repeatedly compare and swap neighbors. Easy to implement. O(n²).
  • Selection sort: repeatedly find the minimum and move it to the front. O(n²).
  • sorted() / sort(): Python's built-in sort. O(n log n), very fast.
  • In real projects, use sorted(). Implementing your own sort is for understanding the concepts!
