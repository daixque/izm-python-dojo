📋 Task

Implement the function get_all_users(conn).

It should retrieve all rows from the users table and return them.

Table contents

  id | name    | age
---+---------+----
 1 | Alice   |  25
 2 | Bob     |  30
 3 | Charlie |  22

Expected return value

  [(1, 'Alice', 25), (2, 'Bob', 30), (3, 'Charlie', 22)]

✏️ Instructions

  • The table creation and data insertion code is already provided
  • Get a cursor with conn.cursor()
  • Execute the query with cursor.execute("SELECT * FROM users")
  • Return all results using cursor.fetchall()

▶ 💡 Hints

  • Basic SELECT structure:
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM users")
    return cursor.fetchall()
  • SELECT * FROM table_name fetches all columns and all rows.
    fetchall() returns the results as a list of tuples.
    Example: [(1, 'Alice', 25), (2, 'Bob', 30)]
