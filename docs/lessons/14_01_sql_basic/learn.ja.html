🗄️ データベースとテーブルとは

データベースとは、たくさんのデータを整理して保存しておく「倉庫」のようなものです。スマートフォンのアプリや Webサービスは、ほぼすべてデータベースを使ってユーザー情報や投稿を管理しています。

データベースの中では、データはテーブル（表）として保存されます。テーブルはExcelのシートのようなもので、行（レコード）と列（カラム）で構成されています。

🔤 SQL とは

SQL（エスキューエル）は、データベースに命令を出すための言語です。データを追加・取得・更新・削除するときに使います。

sqlite3 のインポートと接続
import sqlite3

# メモリ上にデータベースを作成（終了すると消える）
conn = sqlite3.connect(":memory:")
cursor = conn.cursor()
sqlite3 は Python 標準ライブラリなので追加インストール不要です。":memory:" を使うとファイルを作らずメモリ上だけにデータベースを作れます。
テーブルを作成する（CREATE TABLE）
cursor.execute(
"CREATE TABLE users (id INTEGER, name TEXT, age INTEGER)"
)
CREATE TABLE テーブル名 (列名 型, ...) でテーブルを作ります。INTEGER は整数、TEXT は文字列を表す型です。
データを挿入する（INSERT INTO）
cursor.execute(
"INSERT INTO users VALUES (?, ?, ?)",
(1, "Alice", 25)
)
conn.commit()
INSERT INTO テーブル名 VALUES (...) でデータを1行追加します。? はプレースホルダーで、実際の値は第2引数のタプルで渡します。conn.commit() で変更を確定します。
データを取得する（SELECT）
cursor.execute("SELECT * FROM users")
rows = cursor.fetchall()
print(rows)
# [(1, 'Alice', 25), (2, 'Bob', 30)]
SELECT * FROM テーブル名 で全列・全行を取得します。fetchall() は結果をタプルのリストとして返します。

📝 sqlite3 の基本的な流れ

  1. sqlite3.connect(":memory:") でデータベースに接続する
  2. conn.cursor() でカーソルを取得する
  3. cursor.execute(SQL文) でSQL命令を実行する
  4. 変更をする場合は conn.commit() で確定する
  5. データを取得する場合は cursor.fetchall() で結果を得る
