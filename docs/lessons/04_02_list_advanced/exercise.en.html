📋 Task

Implement the function find_max(numbers).

numbers: a non-empty list of numbers

Return the maximum value in the list. Do not use the built-in max() function — instead, use a for loop to compare all elements.

Examples

  find_max([3, 7, 2, 9, 1])    → 9
find_max([5])                → 5
find_max([-5, -2, -10, -1]) → -1

✏️ Instructions

  • The first line max_value = numbers[0] is already provided
  • Use a for loop to examine every element in numbers one by one
  • If the current element is greater than max_value, update max_value with that element
  • After the loop ends, return max_value

▶ 💡 Hints

  • Use a for loop to iterate over all elements:
    for num in numbers:
        # update max_value if num is larger
  • Compare the current element and update the maximum:
    if num > max_value:
        max_value = num
  • The loop variable num holds each element in turn.
    Whenever you find a value larger than max_value, update max_value to track the running maximum.
  • The completed function looks like this:
    def find_max(numbers):
        max_value = numbers[0]
        for num in numbers:
            if num > max_value:
                max_value = num
        return max_value
