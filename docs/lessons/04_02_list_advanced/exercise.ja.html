📋 課題内容

関数 find_max(numbers) を実装しましょう。

numbers: 数値のリスト（空ではありません）

リストの中の最大値を返します。ただし、max() 関数は使わず、for ループで全要素を比較して求めてください。

例

find_max([3, 7, 2, 9, 1])    → 9
find_max([5])                → 5
find_max([-5, -2, -10, -1]) → -1

✏️ やること

  • 関数の最初の行 max_value = numbers[0] はすでに用意されています
  • for ループで numbers のすべての要素を1つずつ確認する
  • 各要素が max_value より大きければ、max_value をその要素で更新する
  • ループが終わったら max_value を返す

▶ 💡 ヒント

  • for ループで全要素を確認します：
    for num in numbers:
        # num が max_value より大きければ更新
  • 現在の要素と最大値を比較して更新します：
    if num > max_value:
        max_value = num
  • ループ変数 num を使って、リストの各要素を順番に見ていきます。
    max_value より大きい値が見つかれば、max_value を更新することで最大値を追跡できます。
  • 完成した関数はこのような形になります：
    def find_max(numbers):
        max_value = numbers[0]
        for num in numbers:
            if num > max_value:
                max_value = num
        return max_value
