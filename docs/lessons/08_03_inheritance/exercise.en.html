📋 Task

Implement a base class Shape and a class Rectangle that inherits from it.

Shape class

  • Has a name attribute (set in __init__)
  • area() method returns 0

Rectangle class (inherits from Shape)

  • Has width and height attributes
  • area() method returns width * height

Example Output

  r = Rectangle(4, 5)
print(r.area())                     # 20
print(isinstance(r, Shape))         # True
print(isinstance(r, Rectangle))     # True

✏️ Instructions

  • Define class Shape: with __init__(self, name) and area(self)
  • Define class Rectangle(Shape): to inherit from Shape
  • In Rectangle's __init__(self, width, height), also call super().__init__("Rectangle") to initialize the parent
  • Rectangle's area() returns self.width * self.height

▶ 💡 Hints

  • Basic Shape class:
    class Shape:
        def __init__(self, name):
            self.name = name
        def area(self):
            return 0
  • Inheriting in Rectangle:
    class Rectangle(Shape):
        def __init__(self, width, height):
            super().__init__("Rectangle")
            self.width = width
            self.height = height
  • Override area() in Rectangle:
    def area(self):
        return self.width * self.height
