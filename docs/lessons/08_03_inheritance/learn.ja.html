継承

📚 説明
English

🧬 継承とは何か

継承（けいしょう）とは、あるクラスの機能を引き継いで、新しいクラスを作る仕組みです。

継承元のクラスを親クラス（スーパークラス）、継承して作ったクラスを子クラス（サブクラス）と呼びます。

子クラスは親クラスのメソッドや属性をそのまま使えます。

継承のイメージ
class Animal:  # 親クラス
    def breathe(self):
        print("呼吸する")

class Dog(Animal):  # Animal を継承
    def bark(self):
        print("ワン！")

dog = Dog()
dog.breathe()  # 親クラスのメソッドが使える
dog.bark()    # 子クラスのメソッド
class Dog(Animal): の (Animal) が「Animal を継承する」という意味です。
Dog クラスは Animal クラスのメソッドをそのまま使えます。

⬆️ super() で親クラスを呼び出す

super() を使うと、子クラスから親クラスのメソッドを呼び出せます。

特に __init__ でよく使います。

super() を使った __init__
class Animal:
    def __init__(self, name):
        self.name = name

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)  # 親の __init__ を呼ぶ
        self.breed = breed

dog = Dog("ポチ", "柴犬")
print(dog.name)   # ポチ（親クラスで設定）
print(dog.breed)  # 柴犬（子クラスで設定）
super().__init__(name) で親クラスの __init__ を呼び出しています。
これで親クラスの初期化処理も実行されます。

🔄 メソッドのオーバーライド

子クラスで親クラスと同じ名前のメソッドを定義すると、子クラスのメソッドで上書き（オーバーライド）されます。

メソッドのオーバーライド
class Shape:
    def area(self):
        return 0

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):  # オーバーライド
        return self.radius ** 2 * 3.14

s = Shape()
print(s.area())        # 0（親クラスの area）
c = Circle(5)
print(c.area())        # 78.5（子クラスの area）
Circle クラスで area() を再定義することで、Shape の area() を上書きしています。
Circle オブジェクトでは上書きした area() が実行されます。
super() でオーバーライドしつつ親も使う
class Animal:
    def describe(self):
        return "動物です"

class Dog(Animal):
    def describe(self):
        parent = super().describe()
        return f"{parent}。犬です。"

dog = Dog()
print(dog.describe())  # 動物です。犬です。
super().describe() で親クラスのメソッドを呼びつつ、追加の処理もできます。

🔍 isinstance() で型を確認する

isinstance(オブジェクト, クラス) で、オブジェクトがそのクラスのインスタンスかどうかを確認できます。

継承関係も考慮されます。

isinstance の使い方
class Animal:
    pass

class Dog(Animal):
    pass

dog = Dog()
print(isinstance(dog, Dog))    # True
print(isinstance(dog, Animal))  # True（継承関係）
print(isinstance(dog, int))    # False
Dog は Animal を継承しているため、Dog のオブジェクトは isinstance(dog, Animal) でも True になります。

💡 まとめ

  • 継承は親クラスの機能を引き継いで新しいクラスを作る仕組み
  • class 子クラス(親クラス): で継承を定義する
  • super() で親クラスのメソッドを呼び出せる
  • オーバーライド: 子クラスで親クラスのメソッドを上書きできる
  • isinstance(obj, Class) でオブジェクトの型を確認できる
