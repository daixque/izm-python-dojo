📋 課題内容

基底クラス Shape と、それを継承した Rectangle クラスを実装しましょう。

Shape クラス

  • name 属性を持つ（__init__ で設定）
  • area() メソッドは 0 を返す

Rectangle クラス（Shape を継承）

  • width（幅）と height（高さ）属性を持つ
  • area() メソッドは width * height を返す

実行例

r = Rectangle(4, 5)
print(r.area())                     # 20
print(isinstance(r, Shape))         # True
print(isinstance(r, Rectangle))     # True

✏️ やること

  • class Shape: を定義して __init__(self, name) と area(self) を実装します
  • class Rectangle(Shape): を定義して Shape を継承します
  • Rectangle の __init__(self, width, height) は super().__init__("Rectangle") で親の初期化も行います
  • Rectangle の area() は self.width * self.height を返します

▶ 💡 ヒント

  • Shape クラスの基本形：
    class Shape:
        def __init__(self, name):
            self.name = name
        def area(self):
            return 0
  • Rectangle クラスの継承：
    class Rectangle(Shape):
        def __init__(self, width, height):
            super().__init__("Rectangle")
            self.width = width
            self.height = height
  • Rectangle の area() をオーバーライドします：
    def area(self):
        return self.width * self.height
