📋 Task

Complete the StudentDB class.

This class uses a SQLite database to manage student names and test scores.

Methods to Implement

  • __init__(self): Create an in-memory database and set up the students table
  • add_student(self, name, scores): Accept a student name and a list of scores, and insert each score into the database
  • get_average(self, name): Return the average score for the given student. Raise ValueError if the student does not exist
  • top_students(self, n): Return a list of (name, average) tuples for the top n students, sorted by average score descending

Example Output

  db = StudentDB()
db.add_student("Alice", [85, 90, 88])
db.add_student("Bob", [70, 75, 80])
db.add_student("Charlie", [95, 92, 98])

print(db.get_average("Alice"))   # 87.666...
print(db.top_students(2))        # [('Charlie', 95.0), ('Alice', 87.666...)]

✏️ Instructions

  • __init__: Connect using sqlite3.connect(":memory:") and execute CREATE TABLE students (name TEXT, score REAL)
  • add_student: Loop through the scores list and insert each score using INSERT INTO students VALUES (?, ?)
  • get_average: Use SELECT AVG(score) FROM students WHERE name = ? to get the average; if the result is None, raise ValueError
  • top_students: Use GROUP BY name ORDER BY avg DESC LIMIT ? to fetch the top n students and return the result of fetchall()

▶ 💡 Hints

  • Example implementation of __init__:
    self.conn = sqlite3.connect(":memory:")
    cursor = self.conn.cursor()
    cursor.execute("CREATE TABLE students (name TEXT, score REAL)")
    self.conn.commit()
  • Example implementation of add_student:
    cursor = self.conn.cursor()
    for score in scores:
        cursor.execute("INSERT INTO students VALUES (?, ?)", (name, score))
    self.conn.commit()
  • Example implementation of get_average:
    cursor = self.conn.cursor()
    cursor.execute("SELECT AVG(score) FROM students WHERE name = ?", (name,))
    result = cursor.fetchone()[0]
    if result is None:
        raise ValueError(f"Student not found: {name}")
    return result
