📋 プロジェクト概要：成績管理システム

今回作るのは、生徒の名前とテストの点数を管理するシステムです。

  • 生徒ごとに複数の点数を登録できる
  • 特定の生徒の平均点を計算できる
  • 平均点が高い順にトップN人のランキングを表示できる
  • 存在しない生徒を検索したときは適切にエラーを出す

これは学校の先生が実際に使いそうなツールです。リアルなシステムを作っているんだという気持ちで取り組みましょう！

② sqlite3（データベース）
import sqlite3

conn = sqlite3.connect(":memory:")
cursor = conn.cursor()
cursor.execute(
    "CREATE TABLE students (name TEXT, score REAL)"
)
conn.commit()
sqlite3 はPythonに標準で付属しているデータベースライブラリです。
":memory:" を使うとデータをメモリ上に保存します（プログラムを終了すると消えます）。
SQL文を使ってデータの追加・検索ができます。

これが「堅牢（けんろう）なプログラム」を作る技術です。

💡 使うSQL文のおさらい

このプロジェクトでは次の3種類のSQL文を使います。

データの挿入（INSERT）
cursor.execute(
    "INSERT INTO students VALUES (?, ?)",
    (name, score)
)
? はプレースホルダーです。第2引数のタプルの値が順番に入ります。
SQLインジェクション攻撃を防ぐために、文字列を直接埋め込まずにプレースホルダーを使うのが正しい書き方です。
平均値の取得（SELECT AVG）
cursor.execute(
    "SELECT AVG(score) FROM students WHERE name = ?",
    (name,)
)
result = cursor.fetchone()[0]
AVG(score) で平均値を計算します。
WHERE name = ? で特定の生徒のデータだけを取得します。
fetchone() は1行だけ取得し、[0] で最初の列の値を取り出します。
ランキングの取得（GROUP BY / ORDER BY）
cursor.execute(
    "SELECT name, AVG(score) as avg FROM students "
    "GROUP BY name ORDER BY avg DESC LIMIT ?",
    (n,)
)
return cursor.fetchall()
GROUP BY name で生徒ごとにまとめます。
ORDER BY avg DESC で平均点の高い順に並べます。
LIMIT ? で上位N件だけ取得します。

🗺️ 設計の考え方：コードを書く前に設計する

大きなプログラムを作るときは、いきなりコードを書き始めるのではなく、まず設計を考えることが大切です。

今回のシステムを設計してみましょう。

  • 何を保存するか → 生徒の名前と点数
  • どんな操作が必要か → 追加・平均取得・ランキング取得
  • エラーはどこで起きるか → 存在しない生徒を検索したとき

この3つを明確にしてからコードを書くと、迷わず進めることができます。
