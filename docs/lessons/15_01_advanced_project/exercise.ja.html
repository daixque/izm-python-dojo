📋 課題内容

StudentDB クラスを完成させましょう。

このクラスは SQLite データベースを使って生徒の名前とテストの点数を管理します。

実装するメソッド

  • __init__(self): メモリ上にデータベースを作り、students テーブルを作成する
  • add_student(self, name, scores): 生徒名と点数のリストを受け取り、1点ずつデータベースに登録する
  • get_average(self, name): 指定した生徒の平均点を返す。生徒が存在しない場合は ValueError を送出する
  • top_students(self, n): 平均点の高い順に上位 n 人の (名前, 平均点) のリストを返す

実行例

  db = StudentDB()
db.add_student("Alice", [85, 90, 88])
db.add_student("Bob", [70, 75, 80])
db.add_student("Charlie", [95, 92, 98])

print(db.get_average("Alice"))   # 87.666...
print(db.top_students(2))        # [('Charlie', 95.0), ('Alice', 87.666...)]

✏️ やること

  • __init__: sqlite3.connect(":memory:") で接続し、CREATE TABLE students (name TEXT, score REAL) を実行してテーブルを作成する
  • add_student: scores のリストをループして、各点数を INSERT INTO students VALUES (?, ?) で登録する
  • get_average: SELECT AVG(score) FROM students WHERE name = ? で平均を取得し、結果が None なら ValueError を raise する
  • top_students: GROUP BY name ORDER BY avg DESC LIMIT ? を使って上位 n 人を取得し、fetchall() の結果を返す

▶ 💡 ヒント

  • __init__ の実装例：
    self.conn = sqlite3.connect(":memory:")
    cursor = self.conn.cursor()
    cursor.execute("CREATE TABLE students (name TEXT, score REAL)")
    self.conn.commit()
  • add_student の実装例：
    cursor = self.conn.cursor()
    for score in scores:
        cursor.execute("INSERT INTO students VALUES (?, ?)", (name, score))
    self.conn.commit()
  • get_average の実装例：
    cursor = self.conn.cursor()
    cursor.execute("SELECT AVG(score) FROM students WHERE name = ?", (name,))
    result = cursor.fetchone()[0]
    if result is None:
        raise ValueError(f"生徒が見つかりません: {name}")
    return result
