📋 課題内容

関数 filter_even(numbers) を実装しましょう。

整数のリスト numbers を受け取り、偶数のみを含む新しいリストを返します。

実行例

  filter_even([1, 2, 3, 4, 5, 6])  # [2, 4, 6]
filter_even([1, 3, 5])            # []
filter_even([10, 15, 20, 25])     # [10, 20]

✏️ やること

  • 空のリスト result = [] はすでに用意されています
  • for ループで numbers の各要素を順番に確認します
  • 偶数かどうかは num % 2 == 0 で判定できます
  • 条件に合う要素は result.append(num) でリストに追加します
  • 最後に return result で結果を返します

▶ 💡 ヒント

  • for ループでリストの全要素を確認します：
    for num in numbers:
        # num が偶数かチェック
  • 偶数の判定には %（余り）演算子を使います：
    if num % 2 == 0:
        # ここで num は偶数
  • 条件に合う要素を result に追加します：
    result.append(num)
  • 完成した関数はこのような形になります：
    for num in numbers:
        if num % 2 == 0:
            result.append(num)
