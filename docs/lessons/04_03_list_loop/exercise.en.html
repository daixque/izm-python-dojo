📋 Task

Implement the function filter_even(numbers).

It receives a list of integers numbers and returns a new list containing only the even numbers.

Examples

  filter_even([1, 2, 3, 4, 5, 6])  # [2, 4, 6]
filter_even([1, 3, 5])            # []
filter_even([10, 15, 20, 25])     # [10, 20]

✏️ Instructions

  • An empty list result = [] is already provided
  • Use a for loop to check each element of numbers in order
  • Check whether a number is even using num % 2 == 0
  • Add matching elements to the list using result.append(num)
  • Return the result at the end with return result

▶ 💡 Hints

  • Use a for loop to go through all elements:
    for num in numbers:
        # check if num is even
  • Use the % (remainder) operator to check for even numbers:
    if num % 2 == 0:
        # num is even here
  • Append matching elements to result:
    result.append(num)
  • The completed function looks like this:
    for num in numbers:
        if num % 2 == 0:
            result.append(num)
