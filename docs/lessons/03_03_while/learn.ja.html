while ループ

📚 説明
English

🔄 while ループとは

for ループは「指定した回数だけ繰り返す」ループでした。

while ループは 条件が真（True）の間、繰り返し続ける ループです。

「何回繰り返すか分からないが、ある条件が満たされるまで繰り返したい」ときに使います。

📝 while ループの基本構文

while ループは次のように書きます。

while ループの基本
count = 0
while count < 5:
    print(count)
    count += 1
# 出力：0, 1, 2, 3, 4
手順：
1. count = 0 で変数を初期化
2. while count < 5: — count が 5 未満の間繰り返す
3. ループ内で count += 1 で count を増やす
4. count が 5 になると条件が偽（False）になりループ終了

重要： count += 1 を忘れると無限ループになります！
while ループの構造
while 条件式:
    繰り返す処理
    条件を変化させる処理
条件式が True の間、インデントされた処理が繰り返されます。
条件が False になるとループを抜けます。

⛔ break 文 - ループを強制終了する

break を使うと、ループの途中で強制的に抜け出せます。

while True:（無限ループ）と組み合わせてよく使われます。

break でループを抜ける
count = 0
while True:
    print(count)
    count += 1
    if count >= 3:
        break
# 出力：0, 1, 2
while True: は永遠に繰り返す無限ループです。
break を実行すると、そこでループが終了します。
count が 3 以上になったとき break でループを抜けています。
数当てゲームのイメージ
secret = 7
while True:
    guess = int(input("数を入力: "))
    if guess == secret:
        print("正解！")
        break
    else:
        print("違います")
正解するまで何度でも入力を受け付けます。
正解したら break でループを終了します。
このように「いつ終わるか分からない繰り返し」に while ループが向いています。

⏭ continue 文 - 次のループへスキップする

continue を使うと、その回の処理をスキップして次のループへ進めます。

奇数だけを表示する（偶数をスキップ）
i = 0
while i < 10:
    i += 1
    if i % 2 == 0:
        continue
    print(i)
# 出力：1, 3, 5, 7, 9
i が偶数のとき continue を実行すると、print(i) をスキップして次のループへ進みます。
結果として奇数だけが表示されます。

⚠️ 無限ループに注意

while ループで条件が永遠に True になり続けると、プログラムが止まらなくなります。

無限ループの例（危険！）
# NG例（実行しないでください）
count = 0
while count < 5:
    print(count)
    # count を増やすのを忘れた！
count が変化しないため、条件が永遠に True のまま。
対策： while ループでは必ず条件を変化させる処理を忘れずに書くこと。

💡 まとめ

  • while 条件: — 条件が True の間繰り返す
  • ループ変数を必ず更新して無限ループを防ぐ
  • break — ループを途中で強制終了する
  • continue — その回の処理をスキップして次のループへ進む
  • while True: + break は「条件を後で判断する」パターン
