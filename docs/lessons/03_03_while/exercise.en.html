📋 Task

Implement the function check_guess(secret, guess).

This function returns a hint for a number guessing game.

  • If guess is less than secret → return "Too low"
  • If guess is greater than secret → return "Too high"
  • If guess equals secret → return "Correct"

Example

  check_guess(50, 30)  → "Too low"
check_guess(50, 70)  → "Too high"
check_guess(50, 50)  → "Correct"

✏️ Instructions

  • The function skeleton is already provided. Use if / elif / else to handle the three cases.
  • If guess < secret, return "Too low"
  • If guess > secret, return "Too high"
  • Otherwise (guess == secret), return "Correct"

▶ 💡 Hints

  • Basic if / elif / else structure:
    if guess < secret:
        return "Too low"
    elif guess > secret:
        return "Too high"
    else:
        return "Correct"
  • guess < secret means "guess is smaller than secret".
    For example, when calling check_guess(50, 30), guess=30 is less than secret=50, so return "Too low".
  • To return a string, write return "string".
    The return values are exactly: "Too low", "Too high", or "Correct" (case-sensitive).
  • The completed function looks like this:
    def check_guess(secret, guess):
        if guess < secret:
            return "Too low"
        elif guess > secret:
            return "Too high"
        else:
            return "Correct"
