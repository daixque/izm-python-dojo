📋 課題内容

関数 check_guess(secret, guess) を実装しましょう。

この関数は数当てゲームのヒントを返します。

  • guess が secret より小さければ → "Too low" を返す
  • guess が secret より大きければ → "Too high" を返す
  • guess が secret と同じなら → "Correct" を返す

実行例

check_guess(50, 30)  → "Too low"
check_guess(50, 70)  → "Too high"
check_guess(50, 50)  → "Correct"

✏️ やること

  • 関数の枠組みは用意済みです。if / elif / else を使って3つの場合に分けて実装してください
  • guess < secret なら "Too low" を返す
  • guess > secret なら "Too high" を返す
  • それ以外（guess == secret）なら "Correct" を返す

▶ 💡 ヒント

  • if / elif / else の基本形：
    if guess < secret:
        return "Too low"
    elif guess > secret:
        return "Too high"
    else:
        return "Correct"
  • guess < secret は「guess が secret より小さい」という条件です。
    例えば check_guess(50, 30) のとき、guess=30 は secret=50 より小さいので "Too low" です。
  • 文字列を返すには return "文字列" と書きます。
    戻り値は "Too low"、"Too high"、"Correct" のいずれかです（大文字・小文字に注意）。
  • 完成した関数はこのような形になります：
    def check_guess(secret, guess):
        if guess < secret:
            return "Too low"
        elif guess > secret:
            return "Too high"
        else:
            return "Correct"
