🔍 Filtering with WHERE

In the previous lesson we retrieved all rows. In practice, you usually want only the rows that match a certain condition. That's what the WHERE clause is for.

Filtering with WHERE
cursor.execute(
"SELECT * FROM users WHERE age >= ?",
(25,)
)
rows = cursor.fetchall()
# [(1, 'Alice', 25), (2, 'Bob', 30)]
WHERE column condition filters rows. >= means "greater than or equal to", = means "equal to", and < means "less than". Pass the ? value as a tuple.
Sorting with ORDER BY
cursor.execute("SELECT * FROM users ORDER BY age")
# ascending order (smallest first)

cursor.execute("SELECT * FROM users ORDER BY age DESC")
# descending order (largest first)
ORDER BY column sorts results in ascending order. Add DESC to sort in descending order.
Aggregate functions (COUNT, AVG, MAX, MIN)
cursor.execute("SELECT COUNT(*) FROM users")
print(cursor.fetchone()) # (3,) → 3 rows

cursor.execute("SELECT AVG(age) FROM users")
print(cursor.fetchone()) # (25.67,) → average age

cursor.execute("SELECT MAX(age), MIN(age) FROM users")
print(cursor.fetchone()) # (30, 22) → max and min
Aggregate functions let you calculate counts, averages, maximums, and minimums in a single SQL statement. fetchone() retrieves only the first row of results.
Grouping with GROUP BY
cursor.execute(
"SELECT age, COUNT(*) FROM users GROUP BY age"
)
rows = cursor.fetchall()
# Returns the count of users for each age
GROUP BY column groups rows with the same value together, used in combination with aggregate functions.

📝 Common SQL Statement Summary

  • SELECT * FROM t WHERE condition → fetch rows matching a condition
  • ORDER BY column → sort ascending (DESC for descending)
  • COUNT(*) → count rows
  • AVG(column) → calculate average
  • MAX(column) / MIN(column) → find maximum / minimum
  • GROUP BY column → aggregate per group
