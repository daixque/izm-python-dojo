WHERE・集計・ORDER BY

📚 説明
English

🔍 WHERE句で絞り込む

前のレッスンでは全件取得しましたが、条件に合うデータだけを取り出したいことが多くあります。そのときは WHERE句 を使います。

WHERE で条件を指定する
cursor.execute(
"SELECT * FROM users WHERE age >= ?",
(25,)
)
rows = cursor.fetchall()
# [(1, 'Alice', 25), (2, 'Bob', 30)]
WHERE 列名 条件 で絞り込みます。>= は「以上」、= は「等しい」、< は「未満」です。? の値はタプルで渡します。
ORDER BY で並べ替える
cursor.execute("SELECT * FROM users ORDER BY age")
# 年齢の昇順

cursor.execute("SELECT * FROM users ORDER BY age DESC")
# 年齢の降順（大きい順）
ORDER BY 列名 で昇順（小さい順）に並べ替えます。DESC をつけると降順（大きい順）になります。
集計関数（COUNT・AVG・MAX・MIN）
cursor.execute("SELECT COUNT(*) FROM users")
print(cursor.fetchone()) # (3,) → 3件

cursor.execute("SELECT AVG(age) FROM users")
print(cursor.fetchone()) # (25.67,) → 平均年齢

cursor.execute("SELECT MAX(age), MIN(age) FROM users")
print(cursor.fetchone()) # (30, 22) → 最大・最小
集計関数を使うと件数・平均・最大値・最小値などを1つのSQL文で求められます。fetchone() は結果の1行目だけを取得します。
GROUP BY でグループ集計する
cursor.execute(
"SELECT age, COUNT(*) FROM users GROUP BY age"
)
rows = cursor.fetchall()
# 各年齢の人数が返される
GROUP BY 列名 で同じ値ごとにグループを作り、集計関数と組み合わせて使います。

📝 よく使うSQL文のまとめ

  • SELECT * FROM t WHERE 条件 → 条件に合うデータを取得
  • ORDER BY 列名 → 昇順に並べ替え（DESC で降順）
  • COUNT(*) → 件数を数える
  • AVG(列名) → 平均値を求める
  • MAX(列名) / MIN(列名) → 最大値・最小値を求める
  • GROUP BY 列名 → グループごとに集計する
