📋 Task

Implement the function get_adult_users(conn, min_age).

It should fetch users from the users table whose age is greater than or equal to min_age, returned in ascending age order.

Examples

get_adult_users(conn, 25)
# → [(1, 'Alice', 25), (2, 'Bob', 30)]

get_adult_users(conn, 30)
# → [(2, 'Bob', 30)]

✏️ Instructions

  • The table creation and data insertion code is already provided
  • Use WHERE age >= ? to filter users by min_age
  • Use ORDER BY age to sort results in ascending order
  • Pass the placeholder value as a tuple: (min_age,)

▶ 💡 Hints

  • Combining WHERE and ORDER BY:
    cursor.execute(
        "SELECT * FROM users WHERE age >= ? ORDER BY age",
        (min_age,)
    )
  • Note the trailing comma in (min_age,).
    This creates a one-element tuple.
    Writing (min_age) without the comma is just parentheses, not a tuple.
