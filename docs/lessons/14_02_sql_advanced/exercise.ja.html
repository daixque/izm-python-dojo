📋 課題内容

関数 get_adult_users(conn, min_age) を実装しましょう。

users テーブルから age が min_age 以上のユーザーを取得し、年齢の昇順で返す関数です。

実行例

  get_adult_users(conn, 25)
# → [(1, 'Alice', 25), (2, 'Bob', 30)]

get_adult_users(conn, 30)
# → [(2, 'Bob', 30)]

✏️ やること

  • テーブルの作成とデータの挿入はすでに用意されています
  • WHERE age >= ? で min_age 以上のユーザーを絞り込みます
  • ORDER BY age で年齢の昇順に並べ替えます
  • ? プレースホルダーに対応する値は (min_age,) のようにタプルで渡します

▶ 💡 ヒント

  • WHERE と ORDER BY を組み合わせた書き方：
    cursor.execute(
        "SELECT * FROM users WHERE age >= ? ORDER BY age",
        (min_age,)
    )
  • (min_age,) の末尾のカンマに注意してください。
    これは要素が1つのタプルを作るためです。
    (min_age) と書くと単なる括弧になってしまいます。
