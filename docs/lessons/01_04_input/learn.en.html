What is the input() function?

So far, we've been using predetermined values in our programs.

The input() function allows you to receive input from users when the program runs.

This enables you to create programs that work with different inputs each time.

Basic usage of input()

The input() function returns the string entered by the user.

To receive input, assign it to a variable:

name = input("What is your name? ")

When the program runs, the message is displayed and waits for input.

When the user enters something and presses Enter, that value is stored in the variable.

Input data is always a string

Important: Data received by input() is always of type string (str).

Even if a number is entered, it's treated as a string.

To use it in calculations, you need to convert it to a number using int() or float().

Receiving numeric input

To receive numeric input and use it in calculations, combine with type conversion:

age = int(input("Enter your age: "))

This stores the entered value as an integer in the variable.

Interactive programs

Using input() allows you to create programs that interact with users.

You can ask questions and process responses accordingly.

For example, you can create a program that asks for a name and returns a personalized greeting.

Code Examples

Basic input
name = input("Enter your name: ")
print("Hello, " + name + "!")
Displays a greeting using the entered name.
Numeric input and calculation
age = int(input("Enter your age: "))
future_age = age + 10
print("In 10 years, you'll be " + str(future_age))
Calculates future age using the entered age.
Multiple inputs
num1 = int(input("First number: "))
num2 = int(input("Second number: "))
total = num1 + num2
print("Sum: " + str(total))
Receives two numbers and calculates their sum.
Decimal input
price = float(input("Product price: "))
tax = price * 0.1
print("Tax: " + str(tax))
Calculates tax based on the entered price.

💡 Tips

  • Write a message inside input() parentheses to tell users what to enter
  • Input is always a string, so remember to convert types when using in calculations
  • Adding a space or ": " at the end of messages makes input easier
  • The program may stop if the user cancels the input
