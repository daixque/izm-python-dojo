📋 課題内容

input()関数を使って、ユーザーに名前を入力してもらい、挨拶を返すプログラムを書きましょう。

「こんにちは、○○さん！」という形式で挨拶を表示してください。

✏️ やること

  • input() を使って、名前を入力してもらいます
  • 入力された名前を変数 name に保存します
  • 「こんにちは、○○さん！」という形式で挨拶を表示します

▶ 💡 ヒント

  • input("名前を入力してください: ") でメッセージと共に入力を受け取れます
  • 文字列の連結には + を使います
  • 例: print("こんにちは、" + name + "さん！")
