⚙️ What is a Function?

When writing programs, you often find yourself repeating the same operations multiple times.

A function is a named block of code that groups a set of operations together.

Once defined, you can call it as many times as you need.

For example, you can create functions like "add two numbers" or "display a greeting" and reuse them throughout your program.

📝 Defining and Calling Functions

In Python, you define a function using the def keyword.

Basic Function Syntax
def function_name():
    # Write your code here
    statement1
    statement2
Key points:
- Write the function name after def
- Add () after the function name
- Add : at the end of the line
- Indent the function body with 4 spaces
Defining and Calling a Function
def say_hello():
    print("Hello!")

say_hello()  # Call the function
say_hello()  # Can be called multiple times
Write say_hello() to call the function.
The power of functions is that you can call them as many times as you need.

🔧 Using Parameters (Arguments)

By adding parameters to a function, you can pass values when calling it.

The values you pass when calling the function are called arguments.

Function with Parameters
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")  # Hello, Alice!
greet("Bob")    # Hello, Bob!
name is the parameter.
When calling the function, you pass values like "Alice" or "Bob" as arguments.
Multiple Parameters
def introduce(name, age):
    print(f"Name: {name}")
    print(f"Age: {age}")

introduce("Alice", 13)
You can define multiple parameters separated by commas.
When calling the function, pass values in the same order.

↩️ The return Statement (Return Value)

The return statement sends a result back to the caller.

This returned value is called the return value.

Function with return
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8
return a + b sends the calculated result back.
You can store the return value in a variable with result = add(3, 5).
Using the Return Value Directly
def multiply(a, b):
    return a * b

# Pass the return value directly to print
print(multiply(4, 3))  # 12
print(multiply(2, 7))  # 14
You can also use the return value directly without storing it in a variable.
print(multiply(4, 3)) is the same as print(12).
Calling a Function Multiple Times
def square(n):
    return n * n

for i in range(1, 6):
    print(f"{i} squared = {square(i)}")
# 1 squared = 1
# 2 squared = 4
# 3 squared = 9
# 4 squared = 16
# 5 squared = 25
You can call a function inside a loop.
Functions let you reuse code efficiently.

🔍 Functions Without a Return Value

Functions without a return statement automatically return None (nothing).

Function Without return
def say_hi():
    print("Hi!")

result = say_hi()  # Hi!
print(result)      # None
The return value of a function without return is None.
None is Python's special value meaning "no value".

💡 Summary

  • Define a function with def function_name():
  • The function body must be indented with 4 spaces
  • Call a function with function_name()
  • Parameters let you pass values when calling the function
  • return sends a result back to the caller
  • Functions without return return None
