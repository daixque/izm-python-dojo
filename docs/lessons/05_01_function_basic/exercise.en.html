📋 Task

Implement the function add(a, b).

The function receives two numbers a and b, and returns their sum using a return statement.

Example Output

  print(add(3, 5))    # 8
print(add(10, 20))  # 30
print(add(-5, 3))   # -2

✏️ Instructions

  • The function definition def add(a, b): is already provided. Implement the function body.
  • Calculate the sum of a and b
  • Use a return statement to return the result

▶ 💡 Hints

  • How to use the return statement:
    def add(a, b):
        return ???
    Replace ??? with an expression that adds a and b.
  • Use the + operator for addition:
    a + b calculates the sum of a and b.
  • The completed function looks like this:
    def add(a, b):
        return a + b
