📋 課題内容

関数 add(a, b) を実装しましょう。

2つの数値 a と b を受け取り、その合計を return 文で返す関数を作ります。

実行例

print(add(3, 5))    # 8
print(add(10, 20))  # 30
print(add(-5, 3))   # -2

✏️ やること

  • 関数定義 def add(a, b): は用意済みです。関数の中身を実装してください
  • a と b を足した結果を計算します
  • return 文を使って計算結果を返します

▶ 💡 ヒント

  • return 文の使い方：
    def add(a, b):
        return ???
    ??? の部分に a と b を足す式を書きます。
  • 足し算は + 演算子を使います：
    a + b で a と b の合計が計算できます。
  • 完成した関数はこのような形になります：
    def add(a, b):
        return a + b
