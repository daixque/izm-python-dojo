📋 課題内容

クラス Counter を実装しましょう。

以下のメソッドを持ちます：

  • increment(): カウントを 1 増やす
  • reset(): カウントを 0 に戻す
  • get_count(): 現在のカウントを返す

実行例

  c = Counter()
c.increment()
c.increment()
print(c.get_count())  # 2
c.reset()
print(c.get_count())  # 0

✏️ やること

  • class Counter: でクラスを定義します
  • __init__ でカウントを 0 に初期化します（self.count = 0）
  • increment(self) で self.count を 1 増やします
  • reset(self) で self.count を 0 に戻します
  • get_count(self) で self.count を返します

▶ 💡 ヒント

  • クラスの構造：
    class Counter:
        def __init__(self):
            self.count = 0
        def increment(self):
            self.count += 1
  • reset と get_count：
    def reset(self):
        self.count = 0
    def get_count(self):
        return self.count
  • self.count += 1 は self.count = self.count + 1 と同じです。
    属性の値を変更するときはこの書き方を使います。
