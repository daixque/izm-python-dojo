📋 Task

Implement the class Counter.

It should have the following methods:

  • increment(): increase the count by 1
  • reset(): reset the count to 0
  • get_count(): return the current count

Example Output

  c = Counter()
c.increment()
c.increment()
print(c.get_count())  # 2
c.reset()
print(c.get_count())  # 0

✏️ Instructions

  • Define the class with class Counter:
  • Initialize the count to 0 in __init__ with self.count = 0
  • In increment(self), increase self.count by 1
  • In reset(self), set self.count back to 0
  • In get_count(self), return self.count

▶ 💡 Hints

  • Class structure:
    class Counter:
        def __init__(self):
            self.count = 0
        def increment(self):
            self.count += 1
  • reset and get_count:
    def reset(self):
        self.count = 0
    def get_count(self):
        return self.count
  • self.count += 1 is the same as self.count = self.count + 1.
    Use this pattern when updating an attribute's value.
