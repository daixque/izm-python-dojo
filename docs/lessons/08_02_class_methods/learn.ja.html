メソッドとインスタンス

📚 説明
English

🔧 インスタンスメソッドとは

クラスの中で定義された関数をメソッドと言います。

特に、オブジェクト（インスタンス）に対して動作するメソッドをインスタンスメソッドと呼びます。

インスタンスメソッドは、必ず第1引数に self を取ります。

インスタンスメソッドの基本
class Counter:
    def __init__(self):
        self.count = 0

    def increment(self):
        self.count += 1

    def get_count(self):
        return self.count

c = Counter()
c.increment()
c.increment()
print(c.get_count())  # 2
increment() メソッドが呼ばれるたびに self.count が 1 増えます。
get_count() で現在のカウントを取得できます。

🎯 self の役割

self はメソッドの中で「このオブジェクト自身」を指します。

self.属性名 でそのオブジェクトの属性にアクセスできます。

self.メソッド名() でそのオブジェクトの他のメソッドを呼び出せます。

self を使った属性の操作
class BankAccount:
    def __init__(self, balance):
        self.balance = balance  # 残高

    def deposit(self, amount):
        self.balance += amount  # 残高を増やす

    def withdraw(self, amount):
        self.balance -= amount  # 残高を減らす

account = BankAccount(1000)
account.deposit(500)
account.withdraw(200)
print(account.balance)  # 1300
self.balance はこのオブジェクトの残高を指します。
deposit や withdraw メソッドで残高を変更できます。
self でメソッドを呼び出す
class Circle:
    def __init__(self, radius):
        self.radius = radius

    def diameter(self):
        return self.radius * 2

    def circumference(self):
        # 他のメソッドを self で呼び出す
        return self.diameter() * 3.14

c = Circle(5)
print(c.diameter())      # 10
print(c.circumference())  # 31.4
メソッドの中から他のメソッドを呼び出すときは self.メソッド名() と書きます。

🔄 属性の更新

メソッドを呼び出すと、オブジェクトの属性を変更できます。

この変更は、同じオブジェクトに対して保持されます。

状態を持つオブジェクト
class Timer:
    def __init__(self):
        self.seconds = 0

    def tick(self):
        self.seconds += 1

    def reset(self):
        self.seconds = 0

t = Timer()
t.tick()
t.tick()
t.tick()
print(t.seconds)  # 3
t.reset()
print(t.seconds)  # 0
tick() を呼ぶたびに seconds が増えます。
reset() で 0 に戻せます。オブジェクトは状態を保持します。

🏠 複数インスタンスの独立性

同じクラスから作ったオブジェクトでも、それぞれは独立した状態を持ちます。

片方のオブジェクトを変更しても、もう片方には影響しません。

インスタンスは独立している
class Counter:
    def __init__(self):
        self.count = 0

    def increment(self):
        self.count += 1

c1 = Counter()
c2 = Counter()

c1.increment()
c1.increment()
c1.increment()

c2.increment()

print(c1.count)  # 3
print(c2.count)  # 1（c1 と独立している）
c1 と c2 は別々のオブジェクトです。
c1 を変更しても、c2 には影響しません。

💡 まとめ

  • インスタンスメソッドは必ず第1引数に self を取る
  • self はメソッド内で「このオブジェクト自身」を指す
  • self.属性名 で属性にアクセス・変更できる
  • self.メソッド名() で同じオブジェクトの他のメソッドを呼べる
  • 同じクラスのインスタンスはそれぞれ独立した状態を持つ
