📋 Task

Let's implement the two core functions of a number guessing game.

Functions to Implement

  • check_guess(secret, guess): A function that judges a guess
  • play_round(secret, guesses): A function that processes a list of guesses

Specification for check_guess

  • If guess is less than secret, return "Too low"
  • If guess is greater than secret, return "Too high"
  • If guess equals secret, return "Correct"

Specification for play_round

  • guesses: a list of guesses (e.g., [30, 70, 50])
  • Check each guess in order using check_guess
  • When "Correct" is found, return a tuple (number of attempts, "Correct")
  • If all guesses are wrong, return (length of guesses, last judgment result)

✏️ Instructions

  • Implement check_guess(secret, guess): use if/elif/else to handle the three cases
  • Implement play_round(secret, guesses): process each guess using a for loop
  • Using enumerate(guesses, 1) makes it easy to track attempt counts starting from 1
  • Return the tuple as soon as "Correct" is found
  • After the loop, return a tuple with the last judgment result if no guess was correct

▶ 💡 Hints

  • Basic structure of check_guess:
    if guess < secret:
        return "Too low"
    elif guess > secret:
        return "Too high"
    else:
        return "Correct"
  • For play_round, using enumerate is helpful:
    for i, guess in enumerate(guesses, 1):
        result = check_guess(secret, guess)
        if result == "Correct":
            return (i, "Correct")
  • After the loop (all guesses were wrong):
    return (len(guesses), check_guess(secret, guesses[-1]))
    guesses[-1] accesses the last element of the list.
  • Expected results for the test runs:
    check_guess(50, 30) → "Too low"
    check_guess(50, 70) → "Too high"
    check_guess(50, 50) → "Correct"
    play_round(50, [30, 70, 50]) → (3, 'Correct')
