📋 課題内容

数当てゲームの中核となる2つの関数を実装しましょう。

実装する関数

  • check_guess(secret, guess)：予想を判定する関数
  • play_round(secret, guesses)：一連の予想リストを処理する関数

check_guess の仕様

  • guess が secret より小さければ "Too low" を返す
  • guess が secret より大きければ "Too high" を返す
  • guess が secret と同じなら "Correct" を返す

play_round の仕様

  • guesses：予想のリスト（例: [30, 70, 50]）
  • 各予想を順に check_guess でチェックする
  • "Correct" になったら (試行回数, "Correct") のタプルを返す
  • 全て外れた場合は (guesses の長さ, 最後の判定結果) を返す

✏️ やること

  • check_guess(secret, guess) を実装する：if/elif/else で3パターンに分岐する
  • play_round(secret, guesses) を実装する：for ループで各予想を処理する
  • enumerate(guesses, 1) を使うと試行回数（1始まり）を簡単に管理できる
  • 正解（"Correct"）が見つかった時点で return でタプルを返す
  • 全て外れた場合は最後の判定結果を含むタプルを返す

▶ 💡 ヒント

  • check_guess の基本構造：
    if guess < secret:
        return "Too low"
    elif guess > secret:
        return "Too high"
    else:
        return "Correct"
  • play_round では enumerate を使うと便利です：
    for i, guess in enumerate(guesses, 1):
        result = check_guess(secret, guess)
        if result == "Correct":
            return (i, "Correct")
  • 全て外れた場合（ループが最後まで回った後）：
    return (len(guesses), check_guess(secret, guesses[-1]))
    guesses[-1] でリストの最後の要素を取得できます。
  • テスト実行の期待結果：
    check_guess(50, 30) → "Too low"
    check_guess(50, 70) → "Too high"
    check_guess(50, 50) → "Correct"
    play_round(50, [30, 70, 50]) → (3, 'Correct')
