📋 Task

Implement the function print_multiplication_table(n) that prints a multiplication table.

It displays the n-times table from 1 to 9.

Output Format

For example, calling print_multiplication_table(3) should produce:

3 × 1 = 3
3 × 2 = 6
3 × 3 = 9
3 × 4 = 12
3 × 5 = 15
3 × 6 = 18
3 × 7 = 21
3 × 8 = 24
3 × 9 = 27

✏️ Instructions

  • The function definition and its call are already provided. Implement the function body.
  • Use a for loop with range(1, 10) to iterate from 1 to 9
  • Print each line in the format f"{n} × {i} = {n * i}"
  • Use × (multiplication sign) and = with single spaces between them

▶ 💡 Hints

  • Basic for loop structure:
    for i in range(1, 10):
        # write the code to repeat here
  • range(1, 10) generates 1, 2, 3, ..., 9 (10 is not included).
    This lets you iterate from 1 to 9.
  • Use a format string to print each line:
    print(f"{n} × {i} = {n * i}")
    n is the table number, i is the loop variable.
  • The completed function looks like this:
    def print_multiplication_table(n):
        for i in range(1, 10):
            print(f"{n} × {i} = {n * i}")
