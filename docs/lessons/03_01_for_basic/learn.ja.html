for ループの基礎

📚 説明
English

🔄 繰り返し処理とは

プログラムでは、同じような処理を何度も繰り返すことがよくあります。

例えば、1から10まで数字を表示したいとき、print(1)、print(2)…と10行書くのは大変です。

ループ（繰り返し）を使えば、短いコードで何度でも同じ処理を実行できます。

📝 for ループの基本構文

for ループは、指定した回数だけ処理を繰り返します。

for ループの基本
for i in range(5):
    print(i)
このコードは 0, 1, 2, 3, 4 と5回出力します。
ポイント：
- i がループ変数（繰り返しのたびに値が変わる）
- range(5) は 0 から 4 までの数を生成
- : の後の処理がインデント（4スペース）で書かれる
for ループの構造
for ループ変数 in range(回数):
    繰り返す処理
重要： インデント（字下げ）が必須です。
繰り返したい処理はすべてインデントして書きます。

🔢 range() 関数の使い方

range() 関数は、数の並びを生成します。使い方は3パターンあります。

range(終了): 0 から始まる
for i in range(3):
    print(i)
# 出力：0, 1, 2
range(3) は 0, 1, 2 を生成します（3は含まれません）。
range(開始, 終了): 開始を指定する
for i in range(1, 4):
    print(i)
# 出力：1, 2, 3
range(1, 4) は 1, 2, 3 を生成します（4は含まれません）。
range(開始, 終了, ステップ): 間隔を指定する
for i in range(0, 10, 2):
    print(i)
# 出力：0, 2, 4, 6, 8
range(0, 10, 2) は 2 おきに 0, 2, 4, 6, 8 を生成します。

🧮 ループ変数を使った計算

ループ変数 i を使って、繰り返しのたびに変わる計算ができます。

1から5の2倍を表示
for i in range(1, 6):
    print(i * 2)
# 出力：2, 4, 6, 8, 10
i が 1, 2, 3, 4, 5 と変わるたびに i * 2 を計算して表示します。
フォーマット文字列との組み合わせ
for i in range(1, 4):
    print(f"{i} × 3 = {i * 3}")
# 出力：
# 1 × 3 = 3
# 2 × 3 = 6
# 3 × 3 = 9
f"..." はフォーマット文字列です。{} の中に変数や式を書くと、その値が埋め込まれます。

📊 実用例 - 九九の表

for ループを使えば、九九の表を簡単に作れます。

3の段の九九
n = 3
for i in range(1, 10):
    print(f"{n} × {i} = {n * i}")
# 出力：
# 3 × 1 = 3
# 3 × 2 = 6
# 3 × 3 = 9
# ...
# 3 × 9 = 27
range(1, 10) で 1 から 9 まで繰り返します（10は含まれません）。
毎回 n × i = n*i の形式で表示されます。

💡 まとめ

  • for i in range(n): で n 回繰り返す
  • range(開始, 終了) で開始と終了を指定できる
  • ループ変数（i）は繰り返しのたびに自動で変わる
  • 繰り返す処理はインデント（4スペース）で書く
  • f"{変数}" でループ変数を文字列に埋め込める
