📋 課題内容

関数 get_element(fruits, index) を実装しましょう。

fruits: 果物名の文字列リスト
index: 取得したい要素の 0 始まりのインデックス

指定されたインデックスの要素を返します。インデックスが範囲外の場合は None を返します。

例

  get_element(["apple", "banana", "orange"], 0)  → "apple"
get_element(["apple", "banana", "orange"], 2)  → "orange"
get_element(["apple", "banana"], 5)            → None（範囲外）

✏️ やること

  • 関数の定義部分は用意済みです。中身を実装してください
  • if 0 <= index < len(fruits): でインデックスが有効な範囲かチェックする
  • 有効なら fruits[index] を返す
  • 範囲外なら None を返す（return None または else: return None）

▶ 💡 ヒント

  • インデックスが有効かどうか確認するには：
    if 0 <= index < len(fruits):
    これで index が 0 以上かつリストの長さ未満かを一度にチェックできます。
  • 条件分岐の構造：
    if 0 <= index < len(fruits):
        return fruits[index]
    else:
        return None
  • len(fruits) はリストの要素数を返します。
    例えば ["apple", "banana", "orange"] なら len() = 3
    有効なインデックスは 0, 1, 2 です（3 は範囲外）。
  • 完成した関数はこのような形になります：
    def get_element(fruits, index):
        if 0 <= index < len(fruits):
            return fruits[index]
        else:
            return None
