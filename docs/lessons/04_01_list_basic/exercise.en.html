📋 Task

Implement the function get_element(fruits, index).

fruits: a list of fruit name strings
index: the zero-based index of the element to retrieve

Return the element at the given index. If the index is out of range, return None.

Examples

  get_element(["apple", "banana", "orange"], 0)  → "apple"
get_element(["apple", "banana", "orange"], 2)  → "orange"
get_element(["apple", "banana"], 5)            → None (out of range)

✏️ Instructions

  • The function definition is already provided. Implement its body.
  • Use if 0 <= index < len(fruits): to check whether the index is within the valid range
  • If valid, return fruits[index]
  • If out of range, return None (use return None or else: return None)

▶ 💡 Hints

  • To check whether the index is valid:
    if 0 <= index < len(fruits):
    This checks that index is at least 0 and less than the length of the list in one step.
  • Structure of the conditional:
    if 0 <= index < len(fruits):
        return fruits[index]
    else:
        return None
  • len(fruits) returns the number of elements in the list.
    For example, ["apple", "banana", "orange"] has len() = 3.
    Valid indexes are 0, 1, 2 (index 3 is out of range).
  • The completed function looks like this:
    def get_element(fruits, index):
        if 0 <= index < len(fruits):
            return fruits[index]
        else:
            return None
