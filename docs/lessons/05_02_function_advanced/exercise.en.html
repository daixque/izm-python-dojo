📋 Task

Implement the function calculate(a, b, operation="add").

It receives two numbers a and b, and an operation type operation, then returns the result of the specified operation.

Operation Types

  • "add": return a + b (default)
  • "subtract": return a - b
  • "multiply": return a * b
  • "divide": return a / b (return None if b is 0)

Example Output

  print(calculate(10, 5))                  # 15 (default is addition)
print(calculate(10, 5, "subtract"))      # 5
print(calculate(10, 5, "multiply"))      # 50
print(calculate(10, 5, "divide"))        # 2.0
print(calculate(10, 0, "divide"))        # None

✏️ Instructions

  • The function signature def calculate(a, b, operation="add"): is already provided
  • Use if / elif to branch based on the value of operation
  • Return the result of each operation using a return statement
  • For division ("divide"), return None when b == 0

▶ 💡 Hints

  • Branch based on the value of operation:
    if operation == "add":
        return ???
    elif operation == "subtract":
        return ???
  • Check for division by zero:
    elif operation == "divide":
        if b == 0:
            return None
        return a / b
  • The four arithmetic operators:
    Addition: a + b
    Subtraction: a - b
    Multiplication: a * b
    Division: a / b
