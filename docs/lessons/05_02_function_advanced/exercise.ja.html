📋 課題内容

関数 calculate(a, b, operation="add") を実装しましょう。

2つの数値 a、b と演算の種類 operation を受け取り、指定された演算の結果を返します。

operationの種類

  • "add": a + b を返す（デフォルト）
  • "subtract": a - b を返す
  • "multiply": a * b を返す
  • "divide": a / b を返す（b が 0 のときは None を返す）

実行例

  print(calculate(10, 5))                  # 15 (デフォルトは加算)
print(calculate(10, 5, "subtract"))      # 5
print(calculate(10, 5, "multiply"))      # 50
print(calculate(10, 5, "divide"))        # 2.0
print(calculate(10, 0, "divide"))        # None

✏️ やること

  • 関数シグネチャ def calculate(a, b, operation="add"): は用意済みです
  • if / elif を使って operation の値に応じた処理を分岐させます
  • 各演算の結果を return 文で返します
  • 除算（"divide"）のとき、b == 0 なら None を返します

▶ 💡 ヒント

  • operation の値に応じて処理を分岐させます：
    if operation == "add":
        return ???
    elif operation == "subtract":
        return ???
  • 除算のときはゼロ除算を確認します：
    elif operation == "divide":
        if b == 0:
            return None
        return a / b
  • 4つの演算子：
    加算: a + b
    減算: a - b
    乗算: a * b
    除算: a / b
