📊 What is Matplotlib?

Matplotlib is a Python library for drawing graphs and charts. It lets you easily create bar charts, line charts, scatter plots, and many other types of visualizations.

Rather than looking at raw numbers, turning data into a chart makes it easy to spot trends and patterns at a glance. This is called data visualization.

🔧 Basic Matplotlib Usage

To use Matplotlib, import matplotlib.pyplot with the alias plt.

How to import
import matplotlib.pyplot as plt
Using as plt means you can write plt.bar() instead of the full name.
Drawing a bar chart
x = [1, 2, 3]
y = [10, 25, 15]

plt.bar(x, y)
plt.show()
plt.bar(x, y) creates a bar chart. x sets the position of each bar and y sets its height. Call plt.show() to display it.
Adding a title and axis labels
plt.bar(x, y)
plt.title("Sales Chart") # chart title
plt.xlabel("Month") # x-axis label
plt.ylabel("Sales (10k)") # y-axis label
plt.show()
Adding labels makes it immediately clear what the chart represents.
Drawing a line chart
x = [1, 2, 3, 4, 5]
y = [10, 15, 12, 20, 18]

plt.plot(x, y)
plt.title("Temperature Changes")
plt.xlabel("Day")
plt.ylabel("Temperature (°C)")
plt.show()
Use plt.plot() instead of plt.bar() to get a line chart. It's great for showing changes over time.

📝 Steps to draw a chart

  1. Import with import matplotlib.pyplot as plt
  2. Prepare your data in variables
  3. Choose a chart type with plt.bar() or plt.plot()
  4. Set labels with plt.title(), plt.xlabel(), plt.ylabel()
  5. Display the chart with plt.show()
