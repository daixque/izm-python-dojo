📋 Task

Complete a program that displays monthly sales data (January to June) as a bar chart.

What to implement

  • Use plt.bar(months, sales) to draw the bar chart
  • Use plt.title("Monthly Sales") to set the chart title
  • Use plt.xlabel("Month") to set the x-axis label
  • Use plt.ylabel("Sales (10k yen)") to set the y-axis label
  • Use plt.show() to display the chart

After running your code, the chart will appear in the "Graph Output" area below the console.

✏️ Instructions

  • Pass x-axis data (months) as the first argument and bar heights (sales) as the second argument to plt.bar(x, y)
  • plt.title(), plt.xlabel(), and plt.ylabel() each take a string argument
  • Always call plt.show() at the end — without it, the chart won't be displayed

▶ 💡 Hints

  • Basic bar chart structure:
    plt.bar(months, sales)
    plt.title("Monthly Sales")
    plt.xlabel("Month")
    plt.ylabel("Sales (10k yen)")
    plt.show()
  • The arguments to plt.bar() are (x_data, heights) in that order.
    Pass months (1–6) for the x-axis and sales (120, 150, ...) for the bar heights.
