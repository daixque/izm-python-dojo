📊 Matplotlib とは

Matplotlib（マットプロットリブ）は、Pythonでグラフや図を描くためのライブラリです。棒グラフ・折れ線グラフ・散布図など、さまざまな種類のグラフを簡単に作れます。

データを「数字の羅列」のまま見るよりも、グラフにすることで傾向や特徴をひと目で把握できます。これをデータの視覚化（ビジュアライゼーション）と呼びます。

🔧 Matplotlib の基本的な使い方

Matplotlib を使うには、まず matplotlib.pyplot を plt という名前でインポートします。

インポートの方法
import matplotlib.pyplot as plt
as plt とすることで、以後 plt.bar() のように短い名前で使えます。
棒グラフを描く
x = [1, 2, 3]
y = [10, 25, 15]

plt.bar(x, y)
plt.show()
plt.bar(x, y) で棒グラフを作成します。x が各棒の位置、y が棒の高さです。plt.show() を呼ぶとグラフが表示されます。
タイトルと軸ラベルを設定する
plt.bar(x, y)
plt.title("売上グラフ") # タイトル
plt.xlabel("月") # x軸ラベル
plt.ylabel("売上（万円）") # y軸ラベル
plt.show()
グラフに説明を加えることで、何を表しているのかが一目でわかるようになります。
折れ線グラフを描く
x = [1, 2, 3, 4, 5]
y = [10, 15, 12, 20, 18]

plt.plot(x, y)
plt.title("気温の変化")
plt.xlabel("日")
plt.ylabel("気温（℃）")
plt.show()
plt.bar() の代わりに plt.plot() を使うと折れ線グラフになります。時間の流れに沿った変化を表すときに便利です。

📝 グラフを描く手順

  1. import matplotlib.pyplot as plt でインポートする
  2. データを変数に用意する
  3. plt.bar() や plt.plot() でグラフの種類を指定する
  4. plt.title()、plt.xlabel()、plt.ylabel() でラベルを設定する
  5. plt.show() でグラフを表示する
