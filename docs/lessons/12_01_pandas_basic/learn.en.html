🐼 What is Pandas?

Python has a powerful library for data analysis called Pandas.

With Pandas, you can easily work with table-style data — just like a spreadsheet in Excel or Google Sheets.

It is perfect for handling data organized in rows and columns, such as school grade sheets, attendance records, or survey results.

To use Pandas, start by writing import pandas as pd at the top of your code.

Importing Pandas
import pandas as pd

# Now we can use the short name "pd"
Writing as pd lets you use the shorter name pd instead of the full pandas.
This is the standard way to import Pandas.

📊 What is a DataFrame?

A DataFrame is the most commonly used data type in Pandas.

Think of it as a table — similar to a spreadsheet in Excel or Google Sheets.

Data is organized into rows (records) and columns.

Creating a DataFrame from a Dictionary
import pandas as pd

data = {
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, 90, 70],
    "grade": ["B", "A", "C"]
}

df = pd.DataFrame(data)
print(df)
The dictionary keys become column names.
The dictionary values (lists) become the data for each column.
The variable name df is commonly used (short for DataFrame).
DataFrame Display
# Output looks like:
# name score grade
# 0 Alice 80 B
# 1 Bob 90 A
# 2 Charlie 70 C
The numbers on the left (0, 1, 2) are the index (row numbers).
The labels on top (name, score, grade) are the column names.

🔍 Selecting Columns and Rows

Let's learn how to extract specific columns and rows from a DataFrame.

Selecting a Column
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, 90, 70]
})

# Extract one column
scores = df["score"]
print(scores)
# 0 80
# 1 90
# 2 70
Use df["column_name"] to extract a specific column.
The result is a Series — a one-dimensional array of data.
Selecting a Row (loc)
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "score": [80, 90, 70]
})

# Get the row at index 1
row = df.loc[1]
print(row)
# name Bob
# score 90
Use df.loc[index] to extract a specific row.
The index starts at 0 (so 0 is the first row).

📈 Useful Methods

DataFrames come with many handy methods for quickly inspecting your data.

head() and tail()
import pandas as pd

df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "Dave", "Eve"],
    "score": [80, 90, 70, 85, 95]
})

print(df.head(3))  # Show first 3 rows
print(df.tail(2))  # Show last 2 rows
head(n): Shows the first n rows (default is 5).
tail(n): Shows the last n rows (default is 5).
Great for peeking at large datasets.
describe()
import pandas as pd

df = pd.DataFrame({
    "score": [80, 90, 70, 85, 95]
})

print(df.describe())
#          score
# count      5.0
# mean      84.0
# std       ....
# min       70.0
# max       95.0
describe() shows statistical summaries for numeric columns at a glance.
It includes count, mean, min, max, and more.
mean() for Calculating Average
import pandas as pd

df = pd.DataFrame({
    "score": [80, 90, 70]
})

avg = df["score"].mean()
print(avg)  # 80.0
df["column_name"].mean() calculates the average of a column.
Other useful methods: .sum() (total), .max() (maximum), .min() (minimum).

💡 Summary

  • Import Pandas with import pandas as pd
  • Create a DataFrame with pd.DataFrame(dictionary)
  • A DataFrame is a table with rows and columns — like a spreadsheet
  • Select a column with df["column_name"]
  • Select a row with df.loc[index]
  • Use head(), tail(), and describe() to quickly inspect data
  • Use methods like .mean() for statistical calculations
