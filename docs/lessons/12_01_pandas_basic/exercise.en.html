📋 Task

Implement the function column_mean(df, column).

This function returns the mean (average) of the specified column in a DataFrame.

Arguments

  • df: A Pandas DataFrame
  • column: The name of the column to average (string)

Example

import pandas as pd
df = pd.DataFrame({"name": ["Alice", "Bob", "Charlie"], "score": [80, 90, 70]})
print(column_mean(df, "score"))  # 80.0

✏️ Instructions

  • Write import pandas as pd at the top of your code
  • Use df[column] to extract the specified column
  • Use Pandas' .mean() method to calculate the average
  • Return the result with a return statement

▶ 💡 Hints

  • To extract a column:
    df[column]
    This gives you a Series (a one-dimensional array of that column's data).
  • Use .mean() to calculate the average:
    df[column].mean()
  • The complete solution is quite simple:
    def column_mean(df, column):
        return df[column].mean()
