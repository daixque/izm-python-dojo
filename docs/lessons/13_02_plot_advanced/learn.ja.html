グラフのカスタマイズ

📚 説明
English

📊 複数のデータ系列を1つのグラフに描く

前のレッスンでは1つのデータを1つのグラフに描きました。実は plt.plot() を複数回呼ぶだけで、1つのグラフに複数のデータ系列を重ねて表示できます。

気温と降水量など、異なるデータを比べたいときに便利です。

複数の折れ線グラフを描く
x = [1, 2, 3, 4, 5]
y1 = [10, 15, 12, 20, 18]
y2 = [5, 8, 7, 11, 9]

plt.plot(x, y1, label="データA")
plt.plot(x, y2, label="データB")
plt.legend()
plt.show()
plt.plot() を2回呼ぶと2本の折れ線が同じグラフに描かれます。label= で各データに名前をつけ、plt.legend() で凡例を表示します。
グリッド線を表示する
plt.plot(x, y1)
plt.grid(True)
plt.show()
plt.grid(True) を呼ぶと方眼紙のようなグリッド線が表示され、値が読み取りやすくなります。
散布図を描く
x = [1, 2, 3, 4, 5]
y = [10, 25, 15, 30, 20]

plt.scatter(x, y)
plt.title("散布図の例")
plt.show()
plt.scatter(x, y) で散布図（てんてんが並ぶグラフ）を描けます。2つの量の関係を見たいときに使います。
色と線のスタイルを変える
plt.plot(x, y1, color="red", linestyle="--", label="赤い破線")
plt.plot(x, y2, color="blue", linestyle="-", label="青い実線")
plt.legend()
plt.show()
color= で色を、linestyle= で線のスタイルを変えられます。"--" は破線、"-" は実線です。複数の系列を区別しやすくなります。

📝 グラフのカスタマイズまとめ

  • plt.plot() を複数回 → 複数の折れ線を重ねる
  • label= 引数 + plt.legend() → 凡例を表示
  • plt.grid(True) → グリッド線を表示
  • plt.scatter(x, y) → 散布図を描く
  • color=、linestyle= → 色・線のスタイルを変える
