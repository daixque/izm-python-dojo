📊 Plotting Multiple Data Series

In the previous lesson you drew one dataset on one chart. By calling plt.plot() more than once, you can overlay multiple data series on the same chart.

This is great for comparing related data — like temperature vs. rainfall.

Drawing multiple line charts
x = [1, 2, 3, 4, 5]
y1 = [10, 15, 12, 20, 18]
y2 = [5, 8, 7, 11, 9]

plt.plot(x, y1, label="Series A")
plt.plot(x, y2, label="Series B")
plt.legend()
plt.show()
Calling plt.plot() twice draws two lines on the same chart. Use label= to name each series, then plt.legend() to show the legend.
Adding gridlines
plt.plot(x, y1)
plt.grid(True)
plt.show()
plt.grid(True) adds graph-paper-style gridlines, making it easier to read exact values.
Drawing a scatter plot
x = [1, 2, 3, 4, 5]
y = [10, 25, 15, 30, 20]

plt.scatter(x, y)
plt.title("Scatter Plot Example")
plt.show()
plt.scatter(x, y) draws a scatter plot — individual dots for each data point. Use it when you want to explore the relationship between two quantities.
Changing colors and line styles
plt.plot(x, y1, color="red", linestyle="--", label="Red dashed")
plt.plot(x, y2, color="blue", linestyle="-", label="Blue solid")
plt.legend()
plt.show()
Use color= to set the color and linestyle= to change the line style. "--" is dashed and "-" is solid. This makes it easier to tell multiple series apart.

📝 Chart Customization Summary

  • Multiple plt.plot() calls → overlay multiple lines
  • label= argument + plt.legend() → show a legend
  • plt.grid(True) → add gridlines
  • plt.scatter(x, y) → draw a scatter plot
  • color=, linestyle= → change color and line style
