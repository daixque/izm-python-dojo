📋 Task

Complete a program that draws both temperature and rainfall data for January through December on a single chart.

What to implement

  • Use plt.plot(months, temperature, label="Temperature (°C)") to draw the temperature line
  • Use plt.plot(months, rainfall, label="Rainfall (mm)") to draw the rainfall line
  • Use plt.title("Monthly Temperature and Rainfall") to set the chart title
  • Use plt.xlabel("Month") to set the x-axis label
  • Use plt.legend() to display the legend
  • Use plt.grid(True) to show gridlines
  • Use plt.show() to display the chart

After running your code, two line charts will appear in the "Graph Output" area.

✏️ Instructions

  • Call plt.plot() twice to overlay multiple data series on the same chart
  • Pass a label= argument to each plt.plot() call — these labels appear in the legend when you call plt.legend()
  • Use plt.grid(True) to add gridlines, making the chart easier to read

▶ 💡 Hints

  • Basic structure for two data series:
    plt.plot(months, temperature, label="Temperature (°C)")
    plt.plot(months, rainfall, label="Rainfall (mm)")
    plt.legend()
  • To add a legend and gridlines:
    plt.legend() # show legend
    plt.grid(True) # show gridlines
    Call these before plt.show().
