📋 Task

Implement the class Animal.

It has two attributes, name and sound, and a speak() method that returns the string "{name} says {sound}".

Example Output

dog = Animal("Dog", "Woof")
print(dog.speak())   # Dog says Woof

cat = Animal("Cat", "Meow")
print(cat.speak())   # Cat says Meow

✏️ Instructions

  • Define the class with class Animal:
  • Initialize the name and sound attributes in __init__(self, name, sound)
  • Define a speak(self) method that returns the string "{name} says {sound}"
  • An f-string like f"{self.name} says {self.sound}" is handy here

  • Basic class structure:

            self.name = ???

            return ???
  • Setting attributes:
    self.name = name
    self.sound = sound
  • Build the return value of speak() with an f-string:
    return f"{self.name} says {self.sound}"
