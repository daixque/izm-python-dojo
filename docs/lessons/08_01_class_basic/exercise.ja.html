📋 課題内容

クラス Animal を実装しましょう。

name（名前）と sound（鳴き声）の2つの属性を持ち、speak() メソッドで「{name}は{sound}と鳴きます」という文字列を返します。

実行例

  dog = Animal("犬", "ワン")
print(dog.speak())   # 犬はワンと鳴きます

cat = Animal("猫", "ニャー")
print(cat.speak())   # 猫はニャーと鳴きます

✏️ やること

  • class Animal: でクラスを定義します
  • __init__(self, name, sound) で name と sound 属性を初期化します
  • speak(self) メソッドを定義して、"{name}は{sound}と鳴きます" という文字列を返します
  • f文字列 f"{self.name}は{self.sound}と鳴きます" を使うと便利です

▶ 💡 ヒント

  • クラスの基本的な形：
    class Animal:
        def __init__(self, name, sound):
            self.name = ???
            self.sound = ???
        def speak(self):
            return ???
  • 属性の設定：
    self.name = name
    self.sound = sound
  • speak() の戻り値はf文字列で作れます：
    return f"{self.name}は{self.sound}と鳴きます"
