📋 課題内容

関数 safe_divide(a, b) を実装しましょう。

a を b で割った結果を返します。

エラーが発生した場合は、エラーの種類を文字列で返します。

実行例

  print(safe_divide(10, 2))    # 5.0
print(safe_divide(10, 0))    # "ZeroDivisionError"
print(safe_divide(10, "a"))  # "TypeError"

✏️ やること

  • 関数シグネチャ def safe_divide(a, b): は用意済みです
  • try: ブロックの中で a / b を計算して返します
  • except ZeroDivisionError: で 0 除算エラーをキャッチし、文字列 "ZeroDivisionError" を返します
  • except TypeError: で型エラーをキャッチし、文字列 "TypeError" を返します

▶ 💡 ヒント

  • 複数の except を使う基本的な形：
    def safe_divide(a, b):
        try:
            return a / b
        except ZeroDivisionError:
            return ???
        except TypeError:
            return ???
  • エラーの種類の名前を文字列として返します：
    except ZeroDivisionError:
        return "ZeroDivisionError"
    except TypeError:
        return "TypeError"
  • 10 / "a" のように、数値でないものを使って割り算すると TypeError が発生します。
    10 / 0 のように 0 で割ると ZeroDivisionError が発生します。
