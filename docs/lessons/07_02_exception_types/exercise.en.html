📋 Task

Implement the function safe_divide(a, b).

It returns the result of dividing a by b.

If an error occurs, return the name of the error type as a string.

Example Output

  print(safe_divide(10, 2))    # 5.0
print(safe_divide(10, 0))    # "ZeroDivisionError"
print(safe_divide(10, "a"))  # "TypeError"

✏️ Instructions

  • The function signature def safe_divide(a, b): is already provided
  • Compute and return a / b inside a try: block
  • Catch division by zero with except ZeroDivisionError: and return the string "ZeroDivisionError"
  • Catch type errors with except TypeError: and return the string "TypeError"

▶ 💡 Hints

  • Basic structure using multiple except clauses:
    def safe_divide(a, b):
        try:
            return a / b
        except ZeroDivisionError:
            return ???
        except TypeError:
            return ???
  • Return the name of the error as a string:
    except ZeroDivisionError:
        return "ZeroDivisionError"
    except TypeError:
        return "TypeError"
  • Dividing with a non-number like 10 / "a" raises a TypeError.
    Dividing by zero like 10 / 0 raises a ZeroDivisionError.
