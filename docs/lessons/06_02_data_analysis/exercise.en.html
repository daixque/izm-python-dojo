📋 Task

Let's implement three functions that analyze student grade data.

Functions to Implement

  • calculate_average(scores): Calculate the average score for each student
  • find_top_student(averages): Return the name of the student with the highest average
  • count_passing(averages, passing_score=60): Return the number of passing students

Specification for calculate_average

  • Argument scores: in the format {"Alice": [85, 90, 88], ...}
  • Return value: in the format {"Alice": 87.666..., ...} (average for each student)
  • Use sum() and len() to calculate the average

Specification for find_top_student

  • Argument averages: in the format {"Alice": 87.67, ...}
  • Return value: the name (string) of the student with the highest average

Specification for count_passing

  • Argument averages: in the format {"Alice": 87.67, ...}
  • Argument passing_score: the passing threshold (default 60)
  • Return value: the number of students with averages at or above the passing score (integer)

✏️ Instructions

  • For calculate_average: loop with scores.items(), divide each list's sum() by its len()
  • For find_top_student: loop with averages.items(), tracking the student name as the maximum is updated
  • For count_passing: loop with averages.values(), increment count when the average is at or above passing_score
  • The default argument passing_score=60 is already defined — no changes needed there

▶ 💡 Hints

  • Structure of calculate_average:
    averages = {}
    for name, score_list in scores.items():
        averages[name] = sum(score_list) / len(score_list)
    return averages
  • Structure of find_top_student:
    top_student = None
    max_avg = -1
    for name, avg in averages.items():
        if avg > max_avg:
            max_avg = avg
            top_student = name
    return top_student
  • Structure of count_passing:
    count = 0
    for avg in averages.values():
        if avg >= passing_score:
            count += 1
    return count
  • Expected results for the test run (using sample data):
    Averages: Alice≈87.67, Bob≈72.33, Charlie≈57.67
    Top student: Alice
    Passing count: 2 (Alice and Bob score 60 or above)
