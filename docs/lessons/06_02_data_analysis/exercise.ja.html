📋 課題内容

生徒の成績データを集計する3つの関数を実装しましょう。

実装する関数

  • calculate_average(scores)：各生徒の平均点を計算する
  • find_top_student(averages)：最高平均点の生徒名を返す
  • count_passing(averages, passing_score=60)：合格者数を返す

calculate_average の仕様

  • 引数 scores：{"Alice": [85, 90, 88], ...} の形式
  • 戻り値：{"Alice": 87.666..., ...} の形式（各生徒の平均値）
  • sum() と len() を使って平均を計算する

find_top_student の仕様

  • 引数 averages：{"Alice": 87.67, ...} の形式
  • 戻り値：最高平均点の生徒の名前（文字列）

count_passing の仕様

  • 引数 averages：{"Alice": 87.67, ...} の形式
  • 引数 passing_score：合格点（デフォルト 60）
  • 戻り値：合格点以上の生徒の数（整数）

✏️ やること

  • calculate_average：scores.items() でループし、各リストの sum() を len() で割る
  • find_top_student：averages.items() でループし、最大値を更新しながら生徒名を追跡する
  • count_passing：averages.values() でループし、passing_score 以上の場合に count を増やす
  • デフォルト引数 passing_score=60 は既に定義されているので変更不要

▶ 💡 ヒント

  • calculate_average の構造：
    averages = {}
    for name, score_list in scores.items():
        averages[name] = sum(score_list) / len(score_list)
    return averages
  • find_top_student の構造：
    top_student = None
    max_avg = -1
    for name, avg in averages.items():
        if avg > max_avg:
            max_avg = avg
            top_student = name
    return top_student
  • count_passing の構造：
    count = 0
    for avg in averages.values():
        if avg >= passing_score:
            count += 1
    return count
  • テスト実行の期待結果（サンプルデータ使用）：
    平均点: Alice≈87.67、Bob≈72.33、Charlie≈57.67
    トップ: Alice
    合格者数: 2（Alice と Bob が60点以上）
