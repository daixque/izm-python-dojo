📋 課題内容

タスク管理アプリを完成させましょう！5つの関数を実装してください。

  1. add_task(tasks, task_name)：タスクをリストに追加して返す
  2. complete_task(tasks, task_id)：指定 id のタスクを完了にして返す
  3. show_tasks(tasks)：タスク一覧を表示する
  4. save_tasks(tasks, filename)：タスクを JSON ファイルに保存する
  5. load_tasks(filename)：JSON ファイルからタスクを読み込む

タスクのデータ構造

各タスクは次の形式の辞書です：{"id": 1, "name": "タスク名", "completed": False}

✏️ やること

  • add_task：新しい id は max([t['id'] for t in tasks], default=0) + 1 で計算します
  • complete_task：ループで id が一致するタスクを探し、completed を True にします
  • show_tasks：完了は [✓] id: name、未完了は [ ] id: name の形式で表示します
  • save_tasks：json.dump(tasks, f, ensure_ascii=False, indent=2) を使います
  • load_tasks：json.load(f) で読み込み、ファイルがなければ空リストを返します
  • ファイルの先頭に import json があります。変更しないでください

▶ 💡 ヒント

  • add_task の新しい id の計算：
    new_id = max([t["id"] for t in tasks], default=0) + 1
    tasks.append({"id": new_id, "name": task_name, "completed": False})
  • complete_task のループ：
    for task in tasks:
        if task["id"] == task_id:
            task["completed"] = True
  • show_tasks の表示：
    status = "✓" if task["completed"] else " "
    print(f"[{status}] {task['id']}: {task['name']}")
  • save_tasks と load_tasks は前のレッスン（ファイル操作）のパターンと同じです。ただし f.write() の代わりに json.dump()、f.read() の代わりに json.load() を使います。
