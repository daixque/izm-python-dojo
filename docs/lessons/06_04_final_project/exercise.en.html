📋 Task

Let's complete the task manager app! Implement the following 5 functions.

  1. add_task(tasks, task_name): Add a task to the list and return it
  2. complete_task(tasks, task_id): Mark the task with the given id as complete and return the list
  3. show_tasks(tasks): Display all tasks
  4. save_tasks(tasks, filename): Save tasks to a JSON file
  5. load_tasks(filename): Load tasks from a JSON file

Task Data Structure

Each task is a dictionary in this format: {"id": 1, "name": "Task name", "completed": False}

✏️ Instructions

  • add_task: Calculate the new id with max([t['id'] for t in tasks], default=0) + 1
  • complete_task: Loop through tasks to find the matching id and set completed to True
  • show_tasks: Display completed tasks as [✓] id: name and incomplete as [ ] id: name
  • save_tasks: Use json.dump(tasks, f, ensure_ascii=False, indent=2)
  • load_tasks: Use json.load(f) to load, return an empty list if the file doesn't exist
  • There is an import json at the top of the file. Do not remove it

▶ 💡 Hints

  • Calculating the new id in add_task:
    new_id = max([t["id"] for t in tasks], default=0) + 1
    tasks.append({"id": new_id, "name": task_name, "completed": False})
  • Loop in complete_task:
    for task in tasks:
        if task["id"] == task_id:
            task["completed"] = True
  • Display in show_tasks:
    status = "✓" if task["completed"] else " "
    print(f"[{status}] {task['id']}: {task['name']}")
  • save_tasks and load_tasks follow the same pattern as the previous file operations lesson. Use json.dump() instead of f.write(), and json.load() instead of f.read().
